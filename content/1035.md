## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1035 Invalid request rewrite (invalid URI path)` occurs when a Cloudflare Transform Rule or URL Rewrite Rule attempts to rewrite the request URI to an invalid path. The rewritten path must be a valid URI path.
