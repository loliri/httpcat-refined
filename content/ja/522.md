## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

HTTP `522 Connection Timed Out` は [Cloudflare](https://cloudflare.com) 固有の非公式 HTTP ステータスコードです。

`522` エラーは、Cloudflare がオリジン Web サーバーへの接続でタイムアウトした場合に発生します。

- 接続確立前：Cloudflare が `SYN` を送信してから 15 秒以内にオリジン Web サーバーが `SYN+ACK` を返さない場合。
- 接続確立後：オリジン Web サーバーが 90 秒以内に Cloudflare のリソースリクエストを確認（`ACK`）しない場合。
