## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

Cloudflare エラー `1019 Compute server error (Worker recursive call)` は、Cloudflare Workers スクリプトが自分自身を再帰的に呼び出し、無限ループが発生した場合に起こります。Worker スクリプトは同じ Worker を再度トリガーするリクエストを生成してはなりません。スクリプトのロジックを見直し、再帰的な呼び出しが発生しないよう修正してください。
