## Descripción

El código de estado HTTP `401 Unauthorized` indica que la solicitud del cliente no se ha completado porque carece de credenciales de autenticación válidas para el recurso solicitado.

Este código de estado se envía con una cabecera de respuesta HTTP `WWW-Authenticate` que contiene información sobre cómo el cliente puede solicitar el recurso de nuevo después de pedir las credenciales de autenticación al usuario.

Este código de estado es similar al código `403 Forbidden`, excepto que en las situaciones que resultan en este código, la autenticación del usuario puede permitir el acceso al recurso.
