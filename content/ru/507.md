## Описание

Код состояния HTTP `507 Insufficient Storage` может быть возвращён в контексте протокола WebDAV (см. RFC 4918).

Он указывает, что метод не мог быть выполнен, поскольку сервер не может сохранить представление, необходимое для успешного выполнения запроса.
