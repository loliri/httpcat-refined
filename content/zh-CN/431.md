## 描述

HTTP `431 Request Header Fields Too Large` 响应状态码表示服务器拒绝处理该请求，因为请求的 HTTP 头过长。可以在缩减请求头大小后重新提交请求。

`431` 可用于请求头总大小过大，或某个单独的头字段过大的情况。为帮助遇到此错误的用户，响应体中应说明是哪种情况，最好还指出哪些头过大，以便用户尝试修复，例如清除 Cookie。

服务器通常在以下情况下返回此状态码：

- `Referer` URL 过长；
- 请求中发送的 Cookie 过多。
