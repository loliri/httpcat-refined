## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP `522 Connection Timed Out` 是 [Cloudflare](https://cloudflare.com) 专用的非官方状态码。

当 Cloudflare 联系源 Web 服务器超时时，会出现 `522` 错误。根据超时发生的时间点，有两种不同的超时情况：

- 连接建立前：Cloudflare 发送 `SYN` 后，源 Web 服务器在 15 秒内未返回 `SYN+ACK`。
- 连接建立后：源 Web 服务器在 90 秒内未确认（`ACK`）Cloudflare 的资源请求。
