## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

HTTP `523 Origin Is Unreachable` — неофициальный код состояния, специфичный для [Cloudflare](https://cloudflare.com).

Ошибка `523` возникает, когда Cloudflare не может связаться с исходным веб-сервером. Обычно это происходит, когда сетевое устройство между Cloudflare и исходным веб-сервером не имеет маршрута к IP-адресу источника.
