## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

HTTP `497 HTTP Request Sent to HTTPS Port` — неофициальный код состояния, специфичный для nginx, возвращаемый сервером, когда обычный HTTP-запрос был отправлен на HTTPS-порт.
