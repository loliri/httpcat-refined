## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

The HTTP response status code `524 A Timeout Occurred` is an unofficial HTTP status code specific to [Cloudflare](https://cloudflare.com).

The `524` error indicates that Cloudflare successfully established a TCP connection to the origin web server, but the origin did not reply with an HTTP response within the 100-second connection timeout.

This differs from `522 Connection Timed Out`, which occurs when the TCP connection itself cannot be established.
