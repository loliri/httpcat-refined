## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP `521 Web Server Is Down` 是 [Cloudflare](https://cloudflare.com) 专用的非官方状态码。

当源 Web 服务器拒绝来自 Cloudflare 的连接时，会出现此错误。

### 参见

- [Cloudflare 文档](https://developers.cloudflare.com/support/troubleshooting/cloudflare-errors/troubleshooting-cloudflare-5xx-errors/#error-521-web-server-is-down)
