## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

Le code HTTP `494 Request Header Too Large` est un code non officiel spécifique à nginx. Il est renvoyé par le serveur pour indiquer que les en-têtes de requête du client dépassent la limite définie par la directive `large_client_header_buffers` dans la configuration nginx.

Cette erreur est similaire au `431 Request Header Fields Too Large` standard, mais `494` est déclenché par nginx lors d'une phase de traitement interne (lors de la lecture des en-têtes de requête) et ne contient généralement pas de corps de réponse.
