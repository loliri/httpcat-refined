## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

Cloudflare 錯誤 `1005 Access Denied (IP banned by origin)` 發生在來源 Web 伺服器封鎖了訪客的 IP 位址時。來源伺服器主動拒絕來自該 IP 的連線，Cloudflare 將此拒絕轉達給客戶端。
