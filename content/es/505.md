## Descripción

El código de estado HTTP `505 HTTP Version Not Supported` indica que la versión de HTTP usada en la solicitud no es compatible con el servidor.

El cuerpo de la respuesta normalmente explica por qué la versión no es compatible y qué protocolos admite el servidor.
