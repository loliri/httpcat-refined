## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1102 Worker exceeded CPU time limit` ocurre cuando un script de Cloudflare Worker excede el límite de tiempo de CPU asignado para su ejecución. Cloudflare Workers tiene límites estrictos de tiempo de CPU para garantizar un uso justo de los recursos entre todos los clientes.
