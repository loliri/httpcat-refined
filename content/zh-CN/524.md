## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP `524 A Timeout Occurred` 是 [Cloudflare](https://cloudflare.com) 专用的非官方状态码。

`524` 错误表示 Cloudflare 已成功与源 Web 服务器建立 TCP 连接，但源服务器在 100 秒的连接超时时间内未返回 HTTP 响应。

这与 `522 Connection Timed Out` 不同——后者发生在 TCP 连接本身无法建立的情况下。
