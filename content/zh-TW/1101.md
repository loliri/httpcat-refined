## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

Cloudflare 錯誤 `1101 Worker threw a JavaScript exception` 發生在 Cloudflare Worker 腳本在執行期間拋出未處理的 JavaScript 例外時。Worker 因腳本中的執行時期錯誤而無法處理請求。
