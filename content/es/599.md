## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

HTTP `599 Network Connect Timeout Error` es un código de estado no oficial usado por algunos proxies para señalar un tiempo de espera de conexión de red detrás del proxy a un cliente delante del proxy.
