## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

Cloudflare 錯誤 `1020 Access denied (Cloudflare Firewall Rule)` 發生在 Cloudflare 防火牆規則（現為 WAF 自訂規則的一部分）封鎖了請求時。網站擁有者可以設定規則，根據 IP 位址、國家、使用者代理、請求標頭等條件封鎖流量。
