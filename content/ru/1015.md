## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

Ошибка Cloudflare `1015 You are being rate limited` возникает, когда посетитель превысил ограничение частоты запросов, настроенное для зоны. Функция ограничения частоты запросов Cloudflare позволяет владельцам сайтов задавать пороговые значения для количества запросов с одного IP-адреса за заданный период времени.
