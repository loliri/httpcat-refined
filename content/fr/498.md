## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

Le code HTTP `498 Invalid Token` est un code non officiel spécifique à ArcGIS, indiquant que la requête HTTP incluait un jeton mais qu'il n'a pas été accepté.
