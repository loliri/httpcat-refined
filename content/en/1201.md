## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1201 Origin returned malformed HTTP` occurs when the origin web server returns an HTTP response that does not conform to the HTTP specification. Cloudflare cannot parse or relay a malformed response to the client.
