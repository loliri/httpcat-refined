## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

Ошибка Cloudflare `1004 Host not configured to serve web traffic` возникает, когда домен существует в системе Cloudflare, но не настроен для обслуживания веб-трафика. Обычно это означает, что домен зарегистрирован в Cloudflare, но веб-хостинг не настроен.
