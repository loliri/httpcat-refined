## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

HTTP `419 Page Expired` は、主に Laravel PHP フレームワークが CSRF トークンの不一致を示すために使用する非公式の HTTP ステータスコードです。
