## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

HTTP `530 Site Frozen` es un código de estado no oficial específico de Cloudflare y Pantheon.

#### Cloudflare

Cloudflare usa el código de estado `530` cuando ocurre un problema entre un usuario y un servidor de sitio web. Acompañado de un código HTTP `1XXX` más detallado, el error depende de la situación específica.

#### Pantheon

Pantheon devuelve el código de estado `530 Site Frozen` cuando un sitio Sandbox inactivo rechaza solicitudes HTTP. Esto se puede resolver reactivando el sitio en el panel de usuario de Pantheon.
