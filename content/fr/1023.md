## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1023 Host not configured (Page Rules)` se produit lorsque le domaine n'est pas correctement configuré dans le système de règles de page de Cloudflare. Cela peut se produire lorsqu'une règle de page fait référence à un nom d'hôte qui n'est pas actif ou correctement configuré dans la zone Cloudflare.
