## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP `520 Web Server Returned an Unknown Error` 是 [Cloudflare](https://cloudflare.com) 专用的非官方状态码。

当源 Web 服务器向 Cloudflare 返回空响应、未知响应或意外响应时，会出现 `520` 错误。这通常是一个兜底错误，用于涵盖 Cloudflare 无法解析源服务器响应的各种情况。

常见原因包括：

- 源服务器在请求处理过程中崩溃或重启；
- 源服务器返回了违反 HTTP 规范的响应；
- 源服务器上的防火墙或安全软件阻止了 Cloudflare 的请求。
