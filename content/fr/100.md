## Description

Le code de statut HTTP `100 Continue` indique que tout s'est bien passé jusqu'ici et que le client doit continuer la requête, ou l'ignorer si elle est déjà terminée.

Pour que le serveur vérifie les en-têtes de la requête, le client doit envoyer `Expect: 100-continue` comme en-tête dans sa requête initiale et recevoir un code de statut `100 Continue` en réponse avant d'envoyer le corps.
