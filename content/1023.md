## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1023 Host not configured (Page Rules)` occurs when the domain is not properly configured in Cloudflare's Page Rules system. This can happen when a Page Rule references a hostname that is not active or properly set up in the Cloudflare zone.
