## Descripción

El código de estado HTTP `303 See Other` indica que las redirecciones no enlazan al recurso solicitado en sí, sino a otra página (como una página de confirmación, una representación de un objeto del mundo real o una página de progreso de subida).

Este código de respuesta se envía a menudo como resultado de una solicitud `PUT` o `POST`. El método utilizado para mostrar esta página redirigida es siempre `GET`.
