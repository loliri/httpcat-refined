## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

Ошибка Cloudflare `1201 Origin returned malformed HTTP` возникает, когда исходный веб-сервер возвращает HTTP-ответ, не соответствующий спецификации HTTP. Cloudflare не может разобрать или передать некорректный ответ клиенту.
