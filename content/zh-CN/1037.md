## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1037 Invalid request rewrite (invalid header name)` 表示某条 Cloudflare 转换规则尝试使用无效的头名称来设置或修改 HTTP 头。头名称必须符合 HTTP 规范。
