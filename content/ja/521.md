## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

HTTP `521 Web Server Is Down` は [Cloudflare](https://cloudflare.com) 固有の非公式 HTTP ステータスコードです。

このエラーは、オリジン Web サーバーが Cloudflare からの接続を拒否した場合に発生します。

### 参照

- [Cloudflare ドキュメント](https://developers.cloudflare.com/support/troubleshooting/cloudflare-errors/troubleshooting-cloudflare-5xx-errors/#error-521-web-server-is-down)
