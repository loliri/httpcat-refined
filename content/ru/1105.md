## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

Ошибка Cloudflare `1105 Rocket Loader exception` возникает, когда функция Rocket Loader Cloudflare сталкивается с ошибкой при обработке JavaScript на странице. Rocket Loader — функция асинхронной загрузки JavaScript для повышения производительности загрузки страниц.
