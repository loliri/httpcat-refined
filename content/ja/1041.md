## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

Cloudflare エラー `1041 Invalid request rewrite (header modification rejected)` は、Transform Rule が保護されたヘッダーを変更しようとした場合に発生します。Cloudflare は特定のシステムヘッダーの変更を禁止しており、それらを対象とした変換ルールは拒否されます。Transform Rule の設定を見直し、変更が許可されているヘッダーのみを対象にしてください。
