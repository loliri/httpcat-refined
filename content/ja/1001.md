## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

Cloudflare エラー `1001 DNS resolution error` は、Cloudflare がオリジン Web サーバーのホスト名を解決できない場合に発生します。これは通常、オリジンの DNS レコードが存在しないか、設定が誤っていることを意味します。DNS レコードが正しく設定されているかどうかを確認してください。
