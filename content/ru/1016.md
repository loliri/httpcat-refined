## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

Ошибка Cloudflare `1016 Origin DNS error` возникает, когда Cloudflare не может разрешить имя хоста исходного веб-сервера. В отличие от ошибки 1001, эта ошибка конкретно указывает на сбой DNS-поиска для исходного сервера в момент запроса.
