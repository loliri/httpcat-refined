## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP `494 Request Header Too Large` 是 nginx 专用的非官方状态码，由服务器返回，表示客户端发送的请求头超出了 nginx 配置中 `large_client_header_buffers` 指令所设定的限制。

此错误与标准的 `431 Request Header Fields Too Large` 类似，但 `494` 是 nginx 在内部处理阶段（读取请求头时）触发的，通常在响应中不携带任何消息体。
