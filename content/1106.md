## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1106 Mirage exception` occurs when Cloudflare's Mirage feature encounters an error. Mirage is an image optimization feature that lazy-loads images and adapts them for mobile devices to improve performance on slower connections.
