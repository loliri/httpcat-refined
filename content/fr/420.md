## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

Les codes de statut HTTP `420 Method Failure` et `420 Enhance your calm` sont des codes non officiels renvoyés par le serveur pour indiquer une erreur client. Envoyé par Spring Framework, il indique qu'une méthode a échoué ; envoyé par Twitter, il indique que le client est limité en débit.

#### Spring Framework

Lorsque Spring Framework envoie le code `420 Method Failure`, cela indique qu'une méthode a échoué. Ce code a été déprécié et n'est plus utilisé par Spring.

#### Twitter

Dans la version 1.0 de l'API Twitter Search and Trends, le code `420 Enhance your calm` était utilisé pour indiquer que le client était limité en débit. Avec la version Standard v1.1 de l'API Twitter, ce statut a été déprécié au profit du `429 Too Many Requests` officiel.
