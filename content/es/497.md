## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

HTTP `497 HTTP Request Sent to HTTPS Port` es un código de estado no oficial específico de nginx, devuelto por el servidor cuando se ha enviado una solicitud HTTP normal al puerto HTTPS.
