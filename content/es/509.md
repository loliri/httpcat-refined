## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

HTTP `509 Bandwidth Limit Exceeded` es un código de estado no oficial específico del servidor web Apache y cPanel, que indica que el cliente del alojamiento web ha excedido la asignación de ancho de banda en el lado del servidor.
