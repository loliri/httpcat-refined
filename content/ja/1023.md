## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

Cloudflare エラー `1023 Host not configured (Page Rules)` は、ドメインが Cloudflare のページルールに正しく設定されていない場合に発生します。ドメインの設定が不完全であるか、ページルールの構成に問題がある可能性があります。Cloudflare ダッシュボードでドメインとページルールの設定を確認してください。
