## Description

Le code de statut HTTP `413 Content Too Large` indique que l'entité de requête est plus grande que les limites définies par le serveur ; le serveur peut fermer la connexion ou retourner un champ d'en-tête `Retry-After`.

Avant RFC 9110, la phrase de réponse pour ce statut était « Payload Too Large ». Ce nom est encore largement utilisé.
