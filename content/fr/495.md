## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

Le code HTTP `495 SSL Certificate Error` est une erreur client non officielle spécifique à nginx, renvoyée par le serveur pour indiquer une erreur lors de la vérification du certificat client.
