## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1000 DNS points to prohibited IP` se produit lorsque le résolveur DNS de Cloudflare détecte que l'enregistrement DNS du domaine pointe vers une adresse IP Cloudflare, ce qui n'est pas autorisé. Cela créerait une boucle où Cloudflare routerait le trafic vers lui-même.
