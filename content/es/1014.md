## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1014 CNAME cross-user banned` ocurre cuando un registro CNAME en una cuenta de Cloudflare apunta a un dominio en otra cuenta de Cloudflare, y este CNAME entre cuentas ha sido bloqueado. Cloudflare restringe ciertas configuraciones CNAME entre usuarios para evitar abusos.
