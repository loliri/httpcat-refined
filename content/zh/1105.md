## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1105 Rocket Loader exception` 表示 Cloudflare 的 Rocket Loader 功能在处理页面上的 JavaScript 时遇到了错误。Rocket Loader 是一项通过异步加载 JavaScript 来提升页面加载性能的功能。
