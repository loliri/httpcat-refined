## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

HTTP `420 Method Failure` 和 `420 Enhance your calm` 是非官方狀態碼，由伺服器返回以表示客戶端錯誤。Spring Framework 傳送時表示方法執行失敗，Twitter 傳送時表示客戶端因請求過多而被限速。

#### Spring Framework

當 Spring Framework 傳送 `420 Method Failure` 時，表示方法執行失敗。此狀態碼已被棄用，Spring 不再使用。

#### Twitter

在 Twitter 搜尋與趨勢 API 1.0 版中，`420 Enhance your calm` 用於表示客戶端因請求過多而被限速，目的是透過限制客戶端在特定時間內的請求數量來緩解流量壓力。隨著 Twitter API Standard v1.1 的發布，此狀態碼已被棄用，改用官方的 `429 Too Many Requests`。
