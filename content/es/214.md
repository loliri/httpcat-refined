## Descripción

<aside class="alert"><strong>Obsoleto:</strong> El código de estado HTTP 214 Transformation Applied y la cabecera HTTP Warning han quedado obsoletos.</aside>

HTTP `214 Transformation Applied` es un código de estado no oficial usado en combinación con la [cabecera HTTP Warning](https://developer.mozilla.org/es/docs/Web/HTTP/Headers/Warning) e incluido en mensajes de respuesta HTTP para indicar que se ha aplicado una transformación a la representación.

Este código de estado y la cabecera `Warning` DEBEN ser añadidos por un proxy si aplica cualquier transformación a la representación, a menos que la cabecera `Warning` ya aparezca en la respuesta.
