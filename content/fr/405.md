## Description

Le code de statut HTTP `405 Method Not Allowed` indique que le serveur connaît la méthode de requête, mais que la ressource cible ne prend pas en charge cette méthode.

Le serveur doit générer un champ d'en-tête `Allow` dans une réponse avec le code de statut `405`. Ce champ doit contenir une liste des méthodes que la ressource cible prend actuellement en charge.
