## 説明

<aside class="alert"><strong>非推奨：</strong>HTTP `214 Transformation Applied` ステータスコードと Warning HTTP ヘッダーは非推奨になりました。</aside>

HTTP `214 Transformation Applied` は、[Warning HTTP ヘッダー](https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Warning)と組み合わせて使用される非公式のステータスコードで、表現に変換が適用されたことを示します。

プロキシが表現に変換（`content-coding`、`media-type` の変更、表現データの変更など）を適用した場合、レスポンスに `Warning` ヘッダーがまだ含まれていなければ、このステータスコードと `Warning` ヘッダーを追加する必要があります。
