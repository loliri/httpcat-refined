## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

Cloudflare エラー `1016 Origin DNS error` は、リクエスト処理時に Cloudflare がオリジンサーバーのホスト名を解決できない場合に発生します。DNS レコードが存在しないか、伝播が完了していない可能性があります。DNS の設定を確認し、オリジンサーバーのレコードが正しく設定されているかどうかを確かめてください。
