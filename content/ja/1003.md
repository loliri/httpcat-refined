## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

Cloudflare エラー `1003 Direct IP access not allowed` は、訪問者が Cloudflare のインフラストラクチャに IP アドレスを直接指定してアクセスしようとした場合に発生します。Cloudflare はドメイン名を通じたアクセスのみを許可しており、IP アドレスへの直接アクセスはサポートされていません。正しいドメイン名を使用してアクセスしてください。
