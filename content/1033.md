## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1033 Argo Tunnel error: connection unavailable` occurs when a Cloudflare Tunnel (formerly Argo Tunnel) connection is unavailable. This means the tunnel connector on the origin server is not running, has crashed, or cannot establish a connection back to Cloudflare's network.
