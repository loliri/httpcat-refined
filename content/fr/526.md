## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

Le code de statut HTTP `526 Invalid SSL Certificate` est un code non officiel spécifique à [Cloudflare](https://cloudflare.com).

L'erreur `526` se produit lorsque Cloudflare ne peut pas valider le certificat SSL sur le serveur web d'origine. Cela se produit généralement dans les cas suivants :

- Le certificat SSL sur l'origine est auto-signé et le mode SSL de Cloudflare est défini sur Full (Strict).
- Le certificat a expiré.
- Le nom d'hôte du certificat ne correspond pas au nom d'hôte de l'origine.
- Le certificat n'est pas approuvé par une autorité de certification publique.
