## Descripción

El código de estado HTTP `407 Proxy Authentication Required` indica que la solicitud no se ha aplicado porque carece de credenciales de autenticación válidas para un servidor proxy situado entre el navegador y el servidor que puede acceder al recurso solicitado.

Este estado se envía con una cabecera `Proxy-Authenticate` que contiene información sobre cómo autorizar correctamente.
