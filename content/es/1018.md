## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1018 Cannot find domain's settings` ocurre cuando Cloudflare no puede localizar los ajustes de configuración del dominio solicitado. Esto puede ocurrir si el dominio se ha añadido recientemente a Cloudflare y los ajustes aún no se han propagado, o si la configuración del dominio está dañada.
