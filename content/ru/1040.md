## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

Ошибка Cloudflare `1040 Invalid request rewrite (transformations disabled)` возникает, когда предпринимается попытка перезаписи запроса, но функция правил преобразования отключена для зоны.
