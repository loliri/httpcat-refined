## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

Le code de statut HTTP `521 Web Server Is Down` est un code non officiel spécifique à [Cloudflare](https://cloudflare.com).

Cette erreur se produit lorsque le serveur web d'origine refuse les connexions de Cloudflare.
