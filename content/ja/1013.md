## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

Cloudflare エラー `1013 HTTP hostname and TLS SNI hostname mismatch` は、リクエストの `Host` ヘッダーと TLS の SNI（Server Name Indication）拡張のホスト名が一致しない場合に発生します。Cloudflare はこれらが一致することを要求しており、不一致の場合はリクエストを拒否します。クライアントまたはプロキシの設定を確認し、ホスト名が正しく送信されているかどうかを確かめてください。
