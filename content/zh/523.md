## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP `523 Origin Is Unreachable` 是 [Cloudflare](https://cloudflare.com) 专用的非官方状态码。

当 Cloudflare 无法联系到源 Web 服务器时，会出现 `523` 错误。通常原因是 Cloudflare 与源服务器之间的某个网络设备没有到达源服务器 IP 地址的路由。
