## Описание

Код состояния HTTP `226 IM Used` в контексте дельта-кодирования устанавливается сервером для указания на то, что он возвращает дельту к полученному GET-запросу. IM расшифровывается как *Instance Manipulations* — термин для алгоритма, генерирующего дельту.

При дельта-кодировании сервер отвечает на `GET`-запросы разницей (дельтой) относительно заданного базового документа. Клиент использует HTTP-заголовок `A-IM:` для указания алгоритма и заголовок `If-None-Match:` для подсказки серверу о последней полученной версии.

<aside class="info"><strong>Примечание:</strong> Браузеры не поддерживают дельта-кодирование с HTTP. Этот код состояния возвращается специализированными серверами для конкретных клиентов.</aside>
