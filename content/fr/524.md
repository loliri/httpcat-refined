## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

Le code de statut HTTP `524 A Timeout Occurred` est un code non officiel spécifique à [Cloudflare](https://cloudflare.com).

L'erreur `524` indique que Cloudflare a établi avec succès une connexion TCP au serveur web d'origine, mais que l'origine n'a pas répondu avec une réponse HTTP dans le délai d'expiration de connexion de 100 secondes.

Cela diffère de `522 Connection Timed Out`, qui se produit lorsque la connexion TCP elle-même ne peut pas être établie.
