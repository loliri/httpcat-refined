## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1034 Edge IP restricted` occurs when the edge IP address used by Cloudflare to connect to the origin is restricted or blocked. This can happen due to firewall rules on the origin server that block Cloudflare's IP ranges.
