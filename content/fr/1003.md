## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1003 Direct IP access not allowed` se produit lorsqu'un visiteur tente d'accéder directement à une adresse IP Cloudflare plutôt que via un nom de domaine. Cloudflare n'autorise pas l'accès IP direct à son infrastructure.
