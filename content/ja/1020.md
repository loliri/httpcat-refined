## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

Cloudflare エラー `1020 Access denied (Cloudflare Firewall Rule)` は、Cloudflare の WAF カスタムルール（旧ファイアウォールルール）によってリクエストがブロックされた場合に発生します。サイトオーナーは IP アドレス、国、ユーザーエージェント、リクエストヘッダーなどの条件に基づいてトラフィックをブロックするルールを設定できます。アクセスが必要な場合は、サイトオーナーに連絡してください。
