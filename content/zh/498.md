## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP `498 Invalid Token` 是 ArcGIS 专用的非官方状态码，表示 HTTP 请求中包含了令牌，但该令牌未被服务器接受。

收到 `498 Invalid Token` 时，表示客户端在请求中提供了令牌，但服务器认为该令牌无效而拒绝接受。
