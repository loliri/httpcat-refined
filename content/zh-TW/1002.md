## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

Cloudflare 錯誤 `1002 DNS points to prohibited IP` 與錯誤 1000 類似，發生在網域的 DNS 記錄解析至 Cloudflare 限制的 IP 位址時，例如回送位址（`127.0.0.1`）或其他禁止的範圍。
