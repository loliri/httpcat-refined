## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

HTTP `524 A Timeout Occurred` es un código de estado no oficial específico de [Cloudflare](https://cloudflare.com).

El error `524` indica que Cloudflare estableció con éxito una conexión TCP con el servidor web de origen, pero el origen no respondió con una respuesta HTTP dentro del tiempo de espera de conexión de 100 segundos.

Esto difiere de `522 Connection Timed Out`, que ocurre cuando la propia conexión TCP no puede establecerse.
