## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP `497 HTTP Request Sent to HTTPS Port` 是 nginx 专用的非官方状态码，由服务器返回，表示客户端向 HTTPS 端口发送了普通的 HTTP 请求。

收到此状态码时，HTTP 请求本身可能是合法的，但服务器拒绝处理，因为它被发送到了 HTTPS 端口。此错误与 `400 Bad Request` 相关。
