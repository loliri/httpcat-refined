## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

HTTP `498 Invalid Token` — неофициальный код состояния, специфичный для ArcGIS, указывающий, что HTTP-запрос содержал токен, но он не был принят.
