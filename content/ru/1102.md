## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

Ошибка Cloudflare `1102 Worker exceeded CPU time limit` возникает, когда скрипт Cloudflare Worker превышает лимит процессорного времени, выделенного для его выполнения. Cloudflare Workers имеет строгие ограничения процессорного времени для обеспечения справедливого использования ресурсов всеми клиентами.
