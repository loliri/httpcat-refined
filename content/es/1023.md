## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1023 Host not configured (Page Rules)` ocurre cuando el dominio no está correctamente configurado en el sistema de reglas de página de Cloudflare. Esto puede ocurrir cuando una regla de página hace referencia a un nombre de host que no está activo o configurado correctamente en la zona de Cloudflare.
