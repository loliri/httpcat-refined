## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1036 Invalid request rewrite (max length exceeded)` 表示某条 Cloudflare 转换规则或 URL 重写规则生成的重写 URL 超出了允许的最大长度。
