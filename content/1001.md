## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1001 DNS resolution error` occurs when Cloudflare cannot resolve the hostname of the origin web server. This typically means the DNS record for the origin does not exist or is misconfigured.
