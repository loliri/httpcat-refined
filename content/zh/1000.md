## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1000 DNS points to prohibited IP` 表示 Cloudflare 的 DNS 解析器检测到该域名的 DNS 记录指向了 Cloudflare 自身的 IP 地址，这是不被允许的。这会造成流量回环，即 Cloudflare 将流量路由回自身。
