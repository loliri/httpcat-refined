## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

HTTP `444 No Response` は nginx 固有の非公式 HTTP ステータスコードで、HTTP 接続が閉じられたことを示します。このステータスコード自体を含め、クライアントには何も送信されません。

悪意のある HTTP リクエスト（不正な `Host` ヘッダーを含むリクエストなど）を処理するのに役立ちます。
