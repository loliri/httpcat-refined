## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1010 Access denied: Browser threat score too high` occurs when Cloudflare's threat intelligence determines that the visitor's browser or IP address has a high threat score, indicating potential malicious activity such as spam, scraping, or bot behavior.
