## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

HTTP `526 Invalid SSL Certificate` は [Cloudflare](https://cloudflare.com) 固有の非公式 HTTP ステータスコードです。

`526` エラーは、Cloudflare がオリジン Web サーバーの SSL 証明書を検証できない場合に発生します。
