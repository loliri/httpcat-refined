## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

Le code de statut HTTP `509 Bandwidth Limit Exceeded` est un code non officiel spécifique au serveur Apache et à cPanel, indiquant que le client d'hébergement web a dépassé l'allocation de bande passante côté serveur.
