## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

HTTP `520 Web Server Returned an Unknown Error` 是 [Cloudflare](https://cloudflare.com) 專用的非官方狀態碼。

當來源 Web 伺服器向 Cloudflare 返回空回應、未知回應或意外回應時，會出現 `520` 錯誤。這通常是一個兜底錯誤，用於涵蓋 Cloudflare 無法解析來源伺服器回應的各種情況。

常見原因包括：

- 來源伺服器在請求處理過程中崩潰或重啟；
- 來源伺服器返回了違反 HTTP 規範的回應；
- 來源伺服器上的防火牆或安全軟體阻止了 Cloudflare 的請求。
