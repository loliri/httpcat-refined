## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

HTTP `499` es un código de estado no oficial específico tanto de ArcGIS (`499 Token Required`) como de nginx (`499 Client Closed Request`).

#### ArcGIS

Cuando se recibe `499 Token Required`, el servidor espera un token en la solicitud HTTP pero no se ha enviado.

#### nginx

Cuando se recibe `499 Client Closed Request`, significa que el cliente ha cerrado la conexión HTTP antes de que el servidor completara la tarea. La respuesta final no puede enviarse y este código solo está disponible en los registros de nginx.
