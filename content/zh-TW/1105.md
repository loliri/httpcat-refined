## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

Cloudflare 錯誤 `1105 Rocket Loader exception` 發生在 Cloudflare 的 Rocket Loader 功能在處理頁面上的 JavaScript 時遇到錯誤。Rocket Loader 是一項非同步載入 JavaScript 以提升頁面載入效能的功能。
