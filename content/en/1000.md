## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1000 DNS points to prohibited IP` occurs when Cloudflare's DNS resolver detects that the domain's DNS record points to a Cloudflare IP address, which is not permitted. This creates a loop where Cloudflare would be routing traffic back to itself.
