## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

Cloudflare 錯誤 `1023 Host not configured (Page Rules)` 發生在網域未在 Cloudflare 的頁面規則系統中正確設定時。當頁面規則引用了在 Cloudflare 區域中未啟用或未正確設定的主機名稱時，可能會發生此情況。
