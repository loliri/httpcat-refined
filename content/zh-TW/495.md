## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

HTTP `495 SSL Certificate Error` 是 nginx 專用的非官方客戶端錯誤，由伺服器返回以表示客戶端憑證驗證發生錯誤。
