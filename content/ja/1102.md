## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

Cloudflare エラー `1102 Worker exceeded CPU time limit` は、Cloudflare Workers スクリプトが許可された CPU 時間の上限を超えた場合に発生します。Workers には実行時間の制限があり、複雑な処理や無限ループによってこの制限を超えることがあります。スクリプトのパフォーマンスを最適化し、処理時間を削減してください。
