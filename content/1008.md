## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1008 Access Denied (ASN banned)` occurs when the Autonomous System Number (ASN) associated with the visitor's IP address has been banned. An ASN represents a collection of IP prefixes managed by a single organization, such as an ISP or hosting provider.
