## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1018 Cannot find domain's settings` se produit lorsque Cloudflare ne peut pas localiser les paramètres de configuration pour le domaine demandé. Cela peut se produire si le domaine a été récemment ajouté à Cloudflare et que les paramètres ne se sont pas encore propagés, ou si la configuration du domaine est corrompue.
