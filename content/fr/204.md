## Description

Le code de statut HTTP `204 No Content` indique qu'une requête a réussi, mais que le client n'a pas besoin de quitter sa page actuelle.

Cela peut être utilisé, par exemple, lors de l'implémentation d'une fonctionnalité « enregistrer et continuer à éditer » pour un site wiki. Dans ce cas, une requête `PUT` serait utilisée pour enregistrer la page, et la réponse `204 No Content` serait envoyée pour indiquer que l'éditeur ne doit pas être remplacé par une autre page.

Une réponse `204` est mise en cache par défaut (un en-tête `ETag` est inclus dans une telle réponse).
