## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1036 Invalid request rewrite (max length exceeded)` occurs when a Cloudflare Transform Rule or URL Rewrite Rule produces a rewritten URL that exceeds the maximum allowed length.
