## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

HTTP `498 Invalid Token` 是 ArcGIS 專用的非官方狀態碼，表示 HTTP 請求中包含了令牌，但該令牌未被接受。

收到 `498 Invalid Token` 狀態碼時，表示客戶端在 HTTP 請求中包含了必要的令牌，但伺服器認為該令牌無效而拒絕接受。
