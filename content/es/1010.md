## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1010 Access denied: Browser threat score too high` ocurre cuando la inteligencia de amenazas de Cloudflare determina que el navegador o la dirección IP del visitante tiene una puntuación de amenaza alta, indicando actividad potencialmente maliciosa como spam, scraping o comportamiento de bot.
