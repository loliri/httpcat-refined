## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

HTTP `523 Origin Is Unreachable` は [Cloudflare](https://cloudflare.com) 固有の非公式 HTTP ステータスコードです。

`523` エラーは、Cloudflare がオリジン Web サーバーに接続できない場合に発生します。通常、Cloudflare とオリジン Web サーバーの間のネットワークデバイスがオリジンの IP アドレスへのルートを持っていない場合に発生します。
