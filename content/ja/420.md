## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

HTTP `420 Method Failure` / `420 Enhance Your Calm` は非公式のステータスコードです。Spring Framework ではメソッドの失敗を示し、Twitter ではクライアントがレート制限されていることを示すために使用されていました。

#### Spring Framework

`420 Method Failure` はメソッドが失敗したことを示します。このステータスコードは Spring で非推奨になり、使用されなくなりました。

#### Twitter

Twitter Search and Trends API バージョン 1.0 では、`420 Enhance Your Calm` はクライアントがレート制限されていることを示すために使用されていました。Standard v1.1 のリリースで非推奨になり、代わりに公式の `429 Too Many Requests` が使用されます。
