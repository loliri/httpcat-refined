## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1014 CNAME cross-user banned` occurs when a CNAME record in one Cloudflare account points to a domain in a different Cloudflare account, and this cross-account CNAME has been banned. Cloudflare restricts certain cross-user CNAME configurations to prevent abuse.
