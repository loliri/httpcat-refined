## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1105 Rocket Loader exception` occurs when Cloudflare's Rocket Loader feature encounters an error while processing JavaScript on the page. Rocket Loader is a feature that asynchronously loads JavaScript to improve page load performance.
