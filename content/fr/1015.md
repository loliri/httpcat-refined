## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1015 You are being rate limited` se produit lorsqu'un visiteur a dépassé la limite de débit configurée pour la zone. La fonctionnalité de limitation de débit de Cloudflare permet aux propriétaires de sites web de définir des seuils pour le nombre de requêtes autorisées depuis une seule adresse IP dans une période donnée.
