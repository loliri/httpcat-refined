## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1000 DNS points to prohibited IP` ocurre cuando el resolutor DNS de Cloudflare detecta que el registro DNS del dominio apunta a una dirección IP de Cloudflare, lo cual no está permitido. Esto crearía un bucle en el que Cloudflare enrutaría el tráfico de vuelta a sí mismo.
