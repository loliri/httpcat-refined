## Descripción

El código de estado HTTP `500 Internal Server Error` indica que el servidor encontró una condición inesperada que le impidió completar la solicitud.

Esta respuesta de error es una respuesta genérica de «cajón de sastre». Normalmente indica que el servidor no encuentra un código de error `5xx` más adecuado. A veces, los administradores de servidor registran respuestas de error como el código `500` con más detalles sobre la solicitud para evitar que el error vuelva a ocurrir en el futuro.
