## Description

Le code de statut HTTP `226 IM Used`, dans le contexte des encodages delta, est un code de statut défini par le serveur pour indiquer qu'il retourne un delta à la requête GET reçue. IM signifie *Instance Manipulations*, le terme utilisé pour décrire un algorithme générant un delta.

Avec l'encodage delta, un serveur répond aux requêtes `GET` avec des différences (appelées deltas) par rapport à un document de base donné. Le client utilise l'en-tête HTTP `A-IM:` pour indiquer l'algorithme de différenciation à utiliser et l'en-tête `If-None-Match:` pour indiquer au serveur la dernière version obtenue.

<aside class="info"><strong>Note :</strong> Les navigateurs ne prennent pas en charge l'encodage delta avec HTTP. Ce code de statut est renvoyé par des serveurs personnalisés utilisés par des clients spécifiques.</aside>
