## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1106 Mirage exception` ocurre cuando la función Mirage de Cloudflare encuentra un error. Mirage es una función de optimización de imágenes que carga imágenes de forma diferida y las adapta para dispositivos móviles para mejorar el rendimiento en conexiones más lentas.
