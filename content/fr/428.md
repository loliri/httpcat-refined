## Description

Le code de statut HTTP `428 Precondition Required` indique que le serveur exige que la requête soit conditionnelle.

Généralement, cela signifie qu'un en-tête de précondition requis, tel que `If-Match`, est manquant.

Lorsqu'un en-tête de précondition ne correspond pas à l'état côté serveur, la réponse doit être `412 Precondition Failed`.
