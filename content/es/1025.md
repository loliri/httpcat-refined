## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1025 Plan limit exceeded` ocurre cuando el plan de Cloudflare del propietario del sitio web no admite la función utilizada, o el uso ha excedido los límites del plan actual.
