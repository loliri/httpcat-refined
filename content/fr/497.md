## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

Le code de statut HTTP `497 HTTP Request Sent to HTTPS Port` est un code non officiel spécifique à nginx, renvoyé par le serveur pour indiquer qu'une requête HTTP normale a été envoyée vers le port HTTPS.
