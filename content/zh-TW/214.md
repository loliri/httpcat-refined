## 描述

<aside class="alert"><strong>已棄用：</strong>此狀態碼已被棄用。</aside>

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

HTTP `214 Transformation Applied` 是非官方狀態碼，與 [Warning HTTP 標頭](https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Warning)搭配使用，包含在 HTTP 回應訊息中，表示已對表示內容套用轉換。

若代理對表示內容套用了任何轉換（如更改 `content-coding`、`media-type` 或修改表示資料），且回應中尚未包含 `Warning` 標頭，則必須加入此狀態碼和 `Warning` 標頭。
