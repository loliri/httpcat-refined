## Description

Le code de statut HTTP `426 Upgrade Required` indique que le serveur refuse d'effectuer la requête en utilisant le protocole actuel mais pourrait être disposé à le faire après que le client se soit mis à niveau vers un protocole différent.

Le serveur envoie un en-tête `Upgrade` avec cette réponse pour indiquer le(s) protocole(s) requis.
