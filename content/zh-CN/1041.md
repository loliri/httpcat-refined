## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1041 Invalid request rewrite (header modification rejected)` 表示某条 Cloudflare 转换规则尝试修改受保护的或只读的 HTTP 头，而这些头是不允许被更改的。
