## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1040 Invalid request rewrite (transformations disabled)` se produit lorsqu'une réécriture de requête est tentée mais que la fonctionnalité Règles de transformation a été désactivée pour la zone.
