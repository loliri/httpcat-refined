## Description

Le code de statut HTTP `508 Loop Detected` peut être donné dans le contexte du protocole WebDAV.

Il indique que le serveur a terminé une opération car il a rencontré une boucle infinie lors du traitement d'une requête avec `"Depth: infinity"`. Ce statut indique que l'opération entière a échoué.
