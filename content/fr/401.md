## Description

Le code de statut HTTP `401 Unauthorized` indique que la requête du client n'a pas été complétée car elle manque d'informations d'authentification valides pour la ressource demandée.

Ce code de statut est envoyé avec un en-tête de réponse HTTP `WWW-Authenticate` qui contient des informations sur la façon dont le client peut demander à nouveau la ressource après avoir demandé les informations d'authentification à l'utilisateur.

Ce code de statut est similaire au code `403 Forbidden`, sauf que dans les situations résultant en ce code, l'authentification de l'utilisateur peut permettre l'accès à la ressource.
