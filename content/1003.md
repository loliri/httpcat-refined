## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1003 Direct IP access not allowed` occurs when a visitor attempts to access a Cloudflare IP address directly rather than through a domain name. Cloudflare does not allow direct IP access to its infrastructure.
