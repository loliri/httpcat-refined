## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1018 Cannot find domain's settings` occurs when Cloudflare cannot locate the configuration settings for the requested domain. This may happen if the domain was recently added to Cloudflare and the settings have not yet propagated, or if the domain configuration is corrupted.
