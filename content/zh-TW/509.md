## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

HTTP `509 Bandwidth Limit Exceeded` 是 Apache Web 伺服器和 cPanel 專用的非官方狀態碼，表示虛擬主機客戶已超過伺服器端的頻寬配額。

收到 `509 Bandwidth Limit Exceeded` 狀態碼時，表示虛擬主機客戶使用的連線或主機方式有頻寬限制。
