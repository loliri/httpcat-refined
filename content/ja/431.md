## 説明

HTTP `431 Request Header Fields Too Large` レスポンスステータスコードは、リクエストの HTTP ヘッダーが長すぎるため、サーバーがリクエストの処理を拒否することを示します。

サーバーは通常、`Referer` URL が長すぎる場合や、リクエストで送信される Cookie が多すぎる場合にこのステータスを返します。
