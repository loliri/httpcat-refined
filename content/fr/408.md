## Description

Le code de statut HTTP `408 Request Timeout` signifie que le serveur souhaite fermer cette connexion inutilisée. Il est envoyé sur une connexion inactive par certains serveurs, même sans aucune requête préalable du client.

Un serveur doit envoyer le champ d'en-tête `Connection: close` dans la réponse, car `408` implique que le serveur a décidé de fermer la connexion plutôt que de continuer à attendre.

<aside class="info"><strong>Note :</strong> Certains serveurs ferment simplement la connexion sans envoyer ce message.</aside>
