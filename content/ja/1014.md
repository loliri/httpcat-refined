## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

Cloudflare エラー `1014 CNAME cross-user banned` は、あるCloudflare アカウントの CNAME レコードが別のアカウントのドメインを指しており、その構成が禁止されている場合に発生します。異なる Cloudflare アカウント間でのこのような CNAME 参照はセキュリティ上の理由から制限されています。同一アカウント内での設定に変更するか、Cloudflare サポートに問い合わせてください。
