## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP `525 SSL Handshake Failed` 是 [Cloudflare](https://cloudflare.com) 专用的非官方状态码。

`525` 错误表示 Cloudflare 与源 Web 服务器之间的 SSL 握手失败。当以下两个条件同时成立时，会出现此错误：

- Cloudflare 与源 Web 服务器之间的 SSL 握手失败；
- 在 Cloudflare SSL/TLS 应用的概览标签中，SSL 模式设置为"完全"或"完全（严格）"。

<aside class="info"><strong>注意：</strong>若托管服务商频繁更改源服务器的 IP 地址，请参阅 Cloudflare 关于动态 DNS 更新的文档。</aside>
