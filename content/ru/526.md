## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

HTTP `526 Invalid SSL Certificate` — неофициальный код состояния, специфичный для [Cloudflare](https://cloudflare.com).

Ошибка `526` возникает, когда Cloudflare не может проверить SSL-сертификат на исходном веб-сервере. Обычно это происходит в следующих случаях:

- SSL-сертификат на источнике является самоподписанным, а режим SSL Cloudflare установлен на Full (Strict).
- Срок действия сертификата истёк.
- Имя хоста в сертификате не совпадает с именем хоста источника.
- Сертификат не является доверенным для публичного центра сертификации.
