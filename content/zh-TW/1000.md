## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

Cloudflare 錯誤 `1000 DNS points to prohibited IP` 發生在 Cloudflare 的 DNS 解析器偵測到網域的 DNS 記錄指向 Cloudflare 自身的 IP 位址時，這是不被允許的，因為這會造成 Cloudflare 將流量路由回自身的迴圈。
