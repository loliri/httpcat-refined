## Descripción

El código de estado HTTP `400 Bad Request` indica que el servidor no puede o no procesará la solicitud debido a algo que se percibe como un error del cliente (por ejemplo, sintaxis de solicitud mal formada, encuadre de mensaje de solicitud inválido o enrutamiento de solicitud engañoso).

<aside class="warning"><strong>Advertencia:</strong> El cliente no debería repetir esta solicitud sin modificación.</aside>
