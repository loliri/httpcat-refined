## 説明

HTTP `207 Multi-Status` レスポンスコードは、複数のステータスが混在する可能性があることを示します。

レスポンスボディは `text/xml` または `application/xml` HTTP エンティティで、`multistatus` ルート要素を持ちます。XML ボディには個々のレスポンスコードが列挙されます。

<aside class="info"><strong>注意：</strong>リソースのコレクションを返す機能は WebDAV プロトコルの一部です。ブラウザで Web ページにアクセスする際にこのステータスコードが表示されることはありません。</aside>
