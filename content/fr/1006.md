## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1006 Access Denied (IP banned)` se produit lorsque l'adresse IP du visiteur a été bannie par Cloudflare lui-même, généralement en raison d'une activité malveillante détectée sur le réseau Cloudflare.
