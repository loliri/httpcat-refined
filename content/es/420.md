## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

Los códigos de estado HTTP `420 Method Failure` y `420 Enhance your calm` son códigos no oficiales devueltos por el servidor para indicar un error del cliente. Cuando los envía Spring Framework indica que un método ha fallado; cuando los envía Twitter indica que el cliente está siendo limitado por exceso de solicitudes.

#### Spring Framework

Cuando Spring Framework envía `420 Method Failure`, indica que un método ha fallado. Este código ha quedado obsoleto y ya no lo usa Spring.

#### Twitter

En la versión 1.0 de la API Twitter Search and Trends, el código `420 Enhance your calm` se usaba para indicar que el cliente estaba siendo limitado por hacer demasiadas solicitudes. Con el lanzamiento de Standard v1.1 de la API de Twitter, este estado quedó obsoleto y se usa en su lugar el oficial `429 Too Many Requests`.
