## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1027 Account suspended` occurs when the Cloudflare account associated with the domain has been suspended, typically due to a violation of Cloudflare's Terms of Service or non-payment.
