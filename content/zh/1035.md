## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1035 Invalid request rewrite (invalid URI path)` 表示某条 Cloudflare 转换规则或 URL 重写规则尝试将请求 URI 重写为无效路径。重写后的路径必须是合法的 URI 路径。
