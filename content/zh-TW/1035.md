## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

Cloudflare 錯誤 `1035 Invalid request rewrite (invalid URI path)` 發生在 Cloudflare 轉換規則或 URL 重寫規則嘗試將請求 URI 重寫為無效路徑時。重寫後的路徑必須是有效的 URI 路徑。
