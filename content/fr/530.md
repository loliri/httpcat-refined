## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

Le code de statut HTTP `530 Site Frozen` est un code non officiel spécifique à Cloudflare et Pantheon.

#### Cloudflare

Cloudflare utilise le code de statut `530` lorsqu'un problème survient entre un utilisateur et un serveur de site web. Accompagné d'un code de statut HTTP `1XXX` plus détaillé, l'erreur dépend de la situation spécifique.

#### Pantheon

Pantheon renvoie le code de statut `530 Site Frozen` lorsqu'un site Sandbox inactif refuse les requêtes HTTP. Cela peut être résolu en réactivant le site dans le tableau de bord utilisateur de Pantheon.
