## Descripción

El código de estado HTTP `403 Forbidden` indica que el servidor entiende la solicitud, pero se niega a autorizarla.

Este estado es similar a `401`, pero para el código `403 Forbidden`, volver a autenticarse no marca diferencia. El acceso está vinculado a la lógica de la aplicación, como permisos insuficientes sobre un recurso.
