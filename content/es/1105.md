## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1105 Rocket Loader exception` ocurre cuando la función Rocket Loader de Cloudflare encuentra un error al procesar JavaScript en la página. Rocket Loader es una función que carga JavaScript de forma asíncrona para mejorar el rendimiento de carga de las páginas.
