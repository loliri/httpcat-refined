## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

Cloudflare 錯誤 `1034 Edge IP restricted` 發生在 Cloudflare 用於連接來源的邊緣 IP 位址被限制或封鎖時。這可能是因為來源伺服器上的防火牆規則封鎖了 Cloudflare 的 IP 範圍。
