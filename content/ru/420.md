## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

Коды состояния HTTP `420 Method Failure` и `420 Enhance your calm` — неофициальные коды, возвращаемые сервером для обозначения ошибки клиента. Spring Framework использует его для обозначения сбоя метода, Twitter — для обозначения ограничения частоты запросов.

#### Spring Framework

Когда Spring Framework отправляет `420 Method Failure`, это означает сбой метода. Этот код устарел и больше не используется Spring.

#### Twitter

В версии 1.0 API Twitter Search and Trends код `420 Enhance your calm` использовался для обозначения ограничения частоты запросов клиента. С выходом Standard v1.1 API Twitter этот статус устарел, и вместо него используется официальный `429 Too Many Requests`.
