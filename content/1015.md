## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1015 You are being rate limited` occurs when a visitor has exceeded the rate limit configured for the zone. Cloudflare's Rate Limiting feature allows website owners to define thresholds for the number of requests allowed from a single IP address within a given time period.
