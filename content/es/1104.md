## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1104 Email Address Obfuscation cannot be processed` ocurre cuando la función de ofuscación de direcciones de correo electrónico de Cloudflare encuentra un error al intentar ofuscar las direcciones de correo electrónico en el cuerpo de la respuesta. Esta función reemplaza las direcciones de correo electrónico con versiones renderizadas con JavaScript para protegerlas de los scrapers.
