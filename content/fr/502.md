## Description

Le code de statut HTTP `502 Bad Gateway` indique que le serveur, agissant comme passerelle ou proxy, a reçu une réponse invalide du serveur en amont.

<aside class="info"><strong>Note :</strong> Une passerelle peut faire référence à différentes choses en réseau et une erreur 502 n'est généralement pas quelque chose que vous pouvez corriger, mais nécessite une correction par le serveur web ou les proxys auxquels vous essayez d'accéder.</aside>
