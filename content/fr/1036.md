## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1036 Invalid request rewrite (max length exceeded)` se produit lorsqu'une règle de transformation Cloudflare ou une règle de réécriture d'URL produit une URL réécrite qui dépasse la longueur maximale autorisée.
