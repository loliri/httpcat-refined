## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

Cloudflare 錯誤 `1008 Access Denied (ASN banned)` 發生在與訪客 IP 位址相關聯的自治系統號碼（ASN）被封鎖時。ASN 代表由單一組織（如 ISP 或主機供應商）管理的一組 IP 前綴。
