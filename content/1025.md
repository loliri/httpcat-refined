## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1025 Plan limit exceeded` occurs when the website owner's Cloudflare plan does not support the feature being used, or the usage has exceeded the limits of their current plan.
