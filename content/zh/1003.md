## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1003 Direct IP access not allowed` 表示访问者尝试直接通过 IP 地址访问 Cloudflare 的基础设施，而非通过域名访问。Cloudflare 不允许直接 IP 访问。
