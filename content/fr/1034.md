## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1034 Edge IP restricted` se produit lorsque l'adresse IP de périphérie utilisée par Cloudflare pour se connecter à l'origine est restreinte ou bloquée. Cela peut se produire en raison de règles de pare-feu sur le serveur d'origine qui bloquent les plages IP de Cloudflare.
