## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

HTTP `499` — неофициальный код состояния, используемый ArcGIS (`499 Token Required`) и nginx (`499 Client Closed Request`).

#### ArcGIS

При получении кода `499 Token Required` сервер ожидает токен в HTTP-запросе, но он не был предоставлен.

#### nginx

При получении кода `499 Client Closed Request` это означает, что клиент закрыл HTTP-соединение до завершения задачи сервером. Финальный ответ не может быть отправлен, и этот код доступен только в журналах nginx.
