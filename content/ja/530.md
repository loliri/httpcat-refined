## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

HTTP `530 Site Frozen` は Cloudflare と Pantheon 固有の非公式 HTTP ステータスコードです。

#### Cloudflare

Cloudflare はユーザーと Web サイトサーバー間で問題が発生した場合に `530` ステータスコードを使用します。より詳細な `1XXX` HTTP ステータスコードが付随します。

#### Pantheon

Pantheon は非アクティブなサンドボックスサイトが HTTP リクエストを拒否した場合に `530 Site Frozen` を返します。Pantheon ユーザーダッシュボードでサイトを再アクティブ化することで解決できます。
