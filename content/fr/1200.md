## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1200 Cache connection limit reached` se produit lorsque le nombre de connexions à la couche de cache de Cloudflare a atteint sa limite maximale. Cela peut se produire lors de pics de trafic, lorsque trop de requêtes simultanées sont traitées.
