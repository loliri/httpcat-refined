## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

Cloudflare 錯誤 `1004 Host not configured to serve web traffic` 發生在網域存在於 Cloudflare 系統中但尚未設定為提供 Web 流量服務時。這通常表示網域已在 Cloudflare 註冊，但尚未設定 Web 主機。
