## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1011 Access denied (Hotlink protection)` occurs when a request is blocked by Cloudflare's Hotlink Protection feature. Hotlink Protection prevents other websites from directly linking to images or other resources on your site, which would consume your bandwidth without your permission.
