## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

HTTP `525 SSL Handshake Failed` es un código de estado no oficial específico de [Cloudflare](https://cloudflare.com).

El error `525` indica que el handshake SSL entre Cloudflare y el servidor web de origen falló. El error `525` ocurre cuando se cumplen estas dos condiciones:

- El handshake SSL falla entre Cloudflare y el servidor web de origen.
- En la pestaña Vista general de la aplicación Cloudflare SSL/TLS está configurado el modo SSL Full o Full (Strict).
