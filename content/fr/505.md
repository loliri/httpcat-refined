## Description

Le code de statut HTTP `505 HTTP Version Not Supported` indique que la version HTTP utilisée dans la requête n'est pas prise en charge par le serveur.

Le corps de la réponse explique généralement pourquoi la version n'est pas prise en charge et quels protocoles le serveur prend en charge.
