## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

HTTP `494 Request Header Too Large` 是 nginx 專用的非官方狀態碼，由伺服器返回以表示客戶端的請求標頭超過了 nginx 設定中 `large_client_header_buffers` 指令所設定的限制。

此錯誤與標準的 `431 Request Header Fields Too Large` 類似，但 `494` 是 nginx 在內部處理階段（讀取請求標頭時）觸發的，通常不包含回應主體。
