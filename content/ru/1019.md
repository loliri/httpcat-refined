## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

Ошибка Cloudflare `1019 Compute server error (Worker recursive call)` возникает, когда скрипт Cloudflare Worker рекурсивно вызывает сам себя, создавая бесконечный цикл. Cloudflare обнаруживает этот рекурсивный паттерн и завершает запрос, чтобы предотвратить исчерпание ресурсов.
