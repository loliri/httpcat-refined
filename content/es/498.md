## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

HTTP `498 Invalid Token` es un código de estado no oficial específico de ArcGIS, que indica que la solicitud HTTP incluía un token pero no fue aceptado.
