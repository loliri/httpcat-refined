## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

HTTP `495 SSL Certificate Error` es un error de cliente no oficial específico de nginx, devuelto por el servidor cuando hay un error en la verificación del certificado del cliente.
