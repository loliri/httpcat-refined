## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

Cloudflare エラー `1036 Invalid request rewrite (max length exceeded)` は、Transform Rule による書き換え後の URL が許可される最大長を超えた場合に発生します。書き換えルールによって生成される URL が長すぎると、このエラーが返されます。Transform Rule の設定を見直し、URL の長さが制限内に収まるよう調整してください。
