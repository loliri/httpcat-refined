## Description

Le code de statut HTTP `301 Moved Permanently` indique que la ressource demandée a été définitivement déplacée vers l'URL indiquée par les en-têtes `Location`. Un navigateur redirige vers la nouvelle URL et les moteurs de recherche mettent à jour leurs liens vers la ressource.

<aside class="info"><strong>Note :</strong> Bien que la spécification exige que la méthode et le corps restent inchangés lors de la redirection, tous les agents utilisateurs ne respectent pas cette exigence. Utilisez le code 301 uniquement comme réponse aux méthodes GET ou HEAD et utilisez le 308 Permanent Redirect pour les méthodes POST à la place.</aside>
