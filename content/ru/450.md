## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

HTTP `450 Blocked by Windows Parental Controls` — неофициальный код состояния, специфичный для Microsoft, возвращаемый сервером для указания на то, что родительский контроль включён и блокирует доступ к запрошенному ресурсу.
