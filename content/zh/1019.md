## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1019 Compute server error (Worker recursive call)` 表示某个 Cloudflare Worker 脚本递归调用了自身，形成无限循环。Cloudflare 检测到此递归模式后会终止请求，以防止资源耗尽。
