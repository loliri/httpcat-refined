## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1104 Email Address Obfuscation cannot be processed` se produit lorsque la fonctionnalité d'obfuscation d'adresses e-mail de Cloudflare rencontre une erreur lors de la tentative d'obfuscation des adresses e-mail dans le corps de la réponse. Cette fonctionnalité remplace les adresses e-mail par des versions rendues en JavaScript pour les protéger des scrapers.
