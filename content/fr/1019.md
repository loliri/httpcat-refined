## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1019 Compute server error (Worker recursive call)` se produit lorsqu'un script Cloudflare Worker s'appelle lui-même de manière récursive, créant une boucle infinie. Cloudflare détecte ce schéma récursif et termine la requête pour éviter l'épuisement des ressources.
