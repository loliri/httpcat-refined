## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

HTTP `450 Blocked by Windows Parental Controls` 是 Microsoft 專用的非官方狀態碼，由伺服器返回以表示家長監護功能已開啟並封鎖了對所請求資源的存取。

收到 `450 Blocked by Windows Parental Controls` 狀態碼時，表示 Windows 家長監護功能已啟用，並拒絕存取目標資源。
