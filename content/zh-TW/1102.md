## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

Cloudflare 錯誤 `1102 Worker exceeded CPU time limit` 發生在 Cloudflare Worker 腳本超過其執行所分配的 CPU 時間限制時。Cloudflare Workers 有嚴格的 CPU 時間限制，以確保所有客戶的資源使用公平。
