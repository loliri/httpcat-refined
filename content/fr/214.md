## Description

<aside class="alert"><strong>Obsolète :</strong> Le code de statut HTTP 214 Transformation Applied et l'en-tête HTTP Warning ont été dépréciés.</aside>

Le code HTTP `214 Transformation Applied` est un code de statut non officiel utilisé en combinaison avec l'[en-tête HTTP Warning](https://developer.mozilla.org/fr/docs/Web/HTTP/Headers/Warning) et inclus dans les messages de réponse HTTP pour indiquer qu'une transformation a été appliquée à la représentation.

Ce code de statut et l'en-tête `Warning` DOIVENT être ajoutés par un proxy s'il applique une transformation à la représentation, sauf si l'en-tête `Warning` figure déjà dans la réponse.
