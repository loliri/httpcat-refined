## Описание

Код состояния HTTP `202 Accepted` указывает, что запрос принят к обработке, но обработка ещё не завершена; фактически она может ещё не начаться.

Запрос может быть в конечном счёте выполнен или не выполнен, поскольку при фактической обработке он может быть отклонён.

`202` не является обязывающим: HTTP не имеет возможности позднее отправить асинхронный ответ с результатом обработки запроса. Он предназначен для случаев, когда запрос обрабатывается другим процессом или сервером, либо для пакетной обработки.
