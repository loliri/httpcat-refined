## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

HTTP `419 Page Expired` — неофициальный код состояния, используемый преимущественно PHP-фреймворком Laravel для обозначения несоответствия CSRF-токена.
