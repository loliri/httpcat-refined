## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

HTTP `523 Origin Is Unreachable` es un código de estado no oficial específico de [Cloudflare](https://cloudflare.com).

El error `523` ocurre cuando Cloudflare no puede contactar con su servidor web de origen. Esto suele suceder cuando un dispositivo de red entre Cloudflare y el servidor web de origen no tiene una ruta a la dirección IP del origen.
