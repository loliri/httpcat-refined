## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1041 Invalid request rewrite (header modification rejected)` se produit lorsqu'une règle de transformation Cloudflare tente de modifier un en-tête HTTP protégé ou en lecture seule qui ne peut pas être modifié.
