## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1014 CNAME cross-user banned` se produit lorsqu'un enregistrement CNAME dans un compte Cloudflare pointe vers un domaine d'un autre compte Cloudflare, et que ce CNAME inter-compte a été banni. Cloudflare restreint certaines configurations CNAME inter-utilisateurs pour prévenir les abus.
