## Descripción

El código de estado HTTP `307 Temporary Redirect` indica que el recurso solicitado ha sido movido temporalmente a la URL indicada por las cabeceras `Location`.

El método y el cuerpo de la solicitud original se reutilizan para realizar la solicitud redirigida. En los casos en que se desee cambiar el método a `GET`, use `303 See Other`.

La única diferencia entre `307` y `302` es que `307` garantiza que el método y el cuerpo no se cambiarán cuando se realice la solicitud redirigida.
