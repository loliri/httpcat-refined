## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP `499` 是一个非官方状态码，在 ArcGIS 中为 `499 Token Required`，在 nginx 中为 `499 Client Closed Request`。

#### ArcGIS

收到 `499 Token Required` 时，表示服务器期望请求中包含令牌，但客户端未提交。

#### nginx

收到 `499 Client Closed Request` 时，表示客户端在服务器完成处理之前已关闭 HTTP 连接，因此无法发送最终响应。此状态码仅出现在 nginx 日志中，不会发送给客户端。
