## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1102 Worker exceeded CPU time limit` 表示某个 Cloudflare Worker 脚本超出了分配给其执行的 CPU 时间限制。Cloudflare Workers 设有严格的 CPU 时间限制，以确保所有客户之间的资源公平使用。
