## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

Cloudflare 錯誤 `1041 Invalid request rewrite (header modification rejected)` 發生在 Cloudflare 轉換規則嘗試修改受保護或唯讀的 HTTP 標頭時，這些標頭無法被更改。
