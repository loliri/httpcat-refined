## 説明

HTTP `505 HTTP Version Not Supported` サーバーエラーレスポンスコードは、リクエストで使用された HTTP バージョンがサーバーでサポートされていないことを示します。

レスポンスボディには通常、そのバージョンがサポートされていない理由と、サーバーがサポートするプロトコルバージョンが説明されています。
