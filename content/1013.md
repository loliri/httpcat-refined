## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1013 HTTP hostname and TLS SNI hostname mismatch` occurs when the hostname provided in the HTTP `Host` header does not match the hostname in the TLS Server Name Indication (SNI) extension. This mismatch can indicate a misconfigured client or a potential security issue.
