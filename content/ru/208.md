## Описание

Код состояния HTTP `208 Already Reported` используется в ответе [207 Multi-Status](https://http.cat/status/207) для экономии места и предотвращения конфликтов. Если один и тот же ресурс запрашивается несколько раз (например, как часть коллекции) с разными путями, только первый сообщается с кодом [200](https://http.cat/status/200). Ответы для всех остальных привязок сообщаются с кодом состояния 208, чтобы не создавать конфликтов и сократить ответ.

<aside class="info"><strong>Примечание:</strong> Возможность привязки ресурса к нескольким путям является расширением протокола WebDAV. Браузеры, обращающиеся к веб-страницам, никогда не встретят этот код состояния.</aside>
