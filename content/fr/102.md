## Description

<aside class="alert"><strong>Obsolète :</strong> Ce code de statut est obsolète. Lorsqu'il est utilisé, les clients peuvent encore l'accepter, mais l'ignoreront simplement.</aside>

Le code de statut HTTP `102 Processing` indique au client que le serveur a reçu la requête complète et est en train de la traiter.

Ce code de statut n'est envoyé que si le serveur s'attend à ce que la requête prenne un temps significatif. Il indique au client que sa requête n'est pas encore abandonnée.
