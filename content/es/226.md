## Descripción

El código de estado HTTP `226 IM Used`, en el contexto de las codificaciones delta, es un código de estado establecido por el servidor para indicar que está devolviendo un delta a la solicitud GET recibida. IM significa *Instance Manipulations*, el término usado para describir un algoritmo que genera un delta.

Con la codificación delta, un servidor responde a las solicitudes `GET` con diferencias (llamadas deltas) respecto a un documento base dado. El cliente usa la cabecera HTTP `A-IM:` para indicar el algoritmo de diferenciación y la cabecera `If-None-Match:` para informar al servidor sobre la última versión obtenida.

<aside class="info"><strong>Nota:</strong> Los navegadores no admiten codificación delta con HTTP. Este código de estado es enviado por servidores personalizados utilizados por clientes específicos.</aside>
