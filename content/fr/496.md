## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

Le code HTTP `496 SSL Certificate Required` est une erreur client non officielle spécifique à nginx, renvoyée par le serveur pour indiquer que le client n'a pas présenté le certificat requis.
