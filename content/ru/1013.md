## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

Ошибка Cloudflare `1013 HTTP hostname and TLS SNI hostname mismatch` возникает, когда имя хоста в HTTP-заголовке `Host` не совпадает с именем хоста в расширении TLS Server Name Indication (SNI). Это несоответствие может указывать на неправильно настроенный клиент или потенциальную проблему безопасности.
