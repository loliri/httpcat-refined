## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1034 Edge IP restricted` ocurre cuando la dirección IP perimetral utilizada por Cloudflare para conectarse al origen está restringida o bloqueada. Esto puede ocurrir debido a reglas de firewall en el servidor de origen que bloquean los rangos de IP de Cloudflare.
