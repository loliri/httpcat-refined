## Description

Le code de statut HTTP `500 Internal Server Error` indique que le serveur a rencontré une condition inattendue qui l'a empêché de satisfaire la requête.

Cette réponse d'erreur est une réponse générique « fourre-tout ». Habituellement, cela indique que le serveur ne peut pas trouver de meilleur code d'erreur `5xx` à renvoyer. Parfois, les administrateurs de serveur consignent les réponses d'erreur comme le code de statut `500` avec plus de détails sur la requête pour empêcher l'erreur de se reproduire à l'avenir.
