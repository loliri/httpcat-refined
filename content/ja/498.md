## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

HTTP `498 Invalid Token` は ArcGIS 固有の非公式 HTTP ステータスコードで、HTTP リクエストにトークンが含まれていたが、サーバーに受け入れられなかったことを示します。
