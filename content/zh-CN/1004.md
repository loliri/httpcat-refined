## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1004 Host not configured to serve web traffic` 表示该域名存在于 Cloudflare 系统中，但尚未配置为提供 Web 流量服务。通常意味着该域名已在 Cloudflare 注册，但尚未设置 Web 托管。
