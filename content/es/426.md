## Descripción

El código de estado HTTP `426 Upgrade Required` indica que el servidor se niega a realizar la solicitud usando el protocolo actual, pero podría estar dispuesto a hacerlo después de que el cliente actualice a un protocolo diferente.

El servidor envía una cabecera `Upgrade` con esta respuesta para indicar el protocolo requerido.
