## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1001 DNS resolution error` se produit lorsque Cloudflare ne peut pas résoudre le nom d'hôte du serveur web d'origine. Cela signifie généralement que l'enregistrement DNS de l'origine n'existe pas ou est mal configuré.
