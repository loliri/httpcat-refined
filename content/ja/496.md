## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

HTTP `496 SSL Certificate Required` は nginx 固有の非公式クライアントエラーステータスコードで、クライアントが必要な証明書を提示しなかったことを示します。
