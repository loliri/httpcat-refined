## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1104 Email Address Obfuscation cannot be processed` occurs when Cloudflare's Email Address Obfuscation feature encounters an error while trying to obfuscate email addresses in the response body. This feature replaces email addresses with JavaScript-rendered versions to protect them from scrapers.
