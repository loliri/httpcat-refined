## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1020 Access denied (Cloudflare Firewall Rule)` se produit lorsqu'une règle de pare-feu Cloudflare (faisant maintenant partie des règles personnalisées WAF) a bloqué la requête. Les propriétaires de sites web peuvent configurer des règles pour bloquer le trafic en fonction de l'adresse IP, du pays, de l'agent utilisateur, des en-têtes de requête et d'autres critères.
