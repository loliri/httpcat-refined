## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1014 CNAME cross-user banned` 表示某个 Cloudflare 账户中的 CNAME 记录指向了另一个 Cloudflare 账户中的域名，且此跨账户 CNAME 已被封禁。Cloudflare 限制某些跨用户 CNAME 配置以防止滥用。
