## Description

Le code de statut HTTP `404 Not Found` indique que le serveur ne peut pas trouver la ressource demandée. Les liens menant à une page `404` sont souvent appelés liens brisés ou morts et peuvent être sujets à la pourriture des liens.

Un code de statut `404` indique uniquement que la ressource est manquante : il ne précise pas si l'absence est temporaire ou permanente. Si une ressource est définitivement supprimée, utilisez plutôt le statut `410 Gone`.
