## 説明

<aside class="alert"><strong>非推奨：</strong>HTTP `305 Use Proxy` ステータスコードはセキュリティ上の懸念から非推奨になり、多くの HTTP クライアントはこのステータスコードを無視します。</aside>

HTTP `305 Use Proxy` は非推奨のステータスコードで、リクエストされたリソースにはプロキシサーバーを通じてのみアクセスできることを示します。
