## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1005 Access Denied (IP banned by origin)` occurs when the origin web server has banned the visitor's IP address. The origin server is actively refusing connections from that IP, and Cloudflare is relaying this refusal.
