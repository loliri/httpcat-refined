## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP `509 Bandwidth Limit Exceeded` 是 Apache Web 服务器和 cPanel 专用的非官方状态码，表示虚拟主机客户端已超出服务器端设定的带宽配额。

收到 `509 Bandwidth Limit Exceeded` 时，表示该虚拟主机客户端使用的连接或托管方式存在带宽限制，且已超出该限制。
