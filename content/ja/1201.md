## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

Cloudflare エラー `1201 Origin returned malformed HTTP` は、オリジンサーバーが HTTP 仕様に準拠していないレスポンスを返した場合に発生します。Cloudflare はオリジンからの不正な形式の HTTP レスポンスを処理できないため、このエラーを返します。オリジンサーバーのアプリケーションやウェブサーバーの設定を確認し、正しい HTTP レスポンスを返すよう修正してください。
