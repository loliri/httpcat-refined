## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

HTTP `419 Page Expired` es un código de estado no oficial usado principalmente por el framework PHP Laravel para indicar un desajuste del token CSRF.
