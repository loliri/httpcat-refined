## Descripción

El código de estado HTTP `424 Failed Dependency` indica que el método no se pudo realizar sobre el recurso porque la acción solicitada dependía de otra acción, y esa acción falló.

Los servidores web normales no suelen devolver este código de estado. Pero otros protocolos, como WebDAV, pueden devolverlo.
