## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1037 Invalid request rewrite (invalid header name)` se produit lorsqu'une règle de transformation Cloudflare tente de définir ou modifier un en-tête HTTP en utilisant un nom d'en-tête invalide. Les noms d'en-têtes doivent être conformes aux spécifications HTTP.
