## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

Cloudflare 錯誤 `1016 Origin DNS error` 發生在 Cloudflare 無法解析來源 Web 伺服器的主機名稱時。與錯誤 1001 不同，此錯誤特指在請求時對來源伺服器進行 DNS 查詢失敗的情況。
