## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

HTTP `524 A Timeout Occurred` は [Cloudflare](https://cloudflare.com) 固有の非公式 HTTP ステータスコードです。

`524` エラーは、Cloudflare がオリジン Web サーバーへの TCP 接続を正常に確立したが、オリジンが 100 秒の接続タイムアウト内に HTTP レスポンスを返さなかったことを示します。
