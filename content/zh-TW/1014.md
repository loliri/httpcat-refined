## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

Cloudflare 錯誤 `1014 CNAME cross-user banned` 發生在一個 Cloudflare 帳戶中的 CNAME 記錄指向另一個 Cloudflare 帳戶中的網域，且此跨帳戶 CNAME 已被封鎖時。Cloudflare 限制某些跨使用者的 CNAME 設定以防止濫用。
