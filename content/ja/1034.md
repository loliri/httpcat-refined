## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

Cloudflare エラー `1034 Edge IP restricted` は、Cloudflare がオリジンサーバーへの接続に使用するエッジ IP アドレスが制限またはブロックされている場合に発生します。オリジンサーバーのファイアウォールが Cloudflare の IP 範囲からのトラフィックを拒否している可能性があります。Cloudflare が公開している IP アドレス範囲をオリジンサーバーのファイアウォールで許可してください。
