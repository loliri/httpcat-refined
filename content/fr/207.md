## Description

Le code de réponse HTTP `207 Multi-Status` indique qu'il peut y avoir un mélange de réponses.

Le corps de la réponse est une entité HTTP `text/xml` ou `application/xml` avec un élément racine `multistatus`. Le corps XML listera tous les codes de réponse individuels.

<aside class="info"><strong>Note :</strong> La capacité à retourner une collection de ressources fait partie du protocole WebDAV (elle peut être reçue par des applications web accédant à un serveur WebDAV). Les navigateurs accédant à des pages web ne rencontreront jamais ce code de statut.</aside>
