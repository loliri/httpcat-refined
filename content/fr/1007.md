## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1007 Access Denied (Self-signed SSL prohibited)` se produit lorsque le serveur d'origine présente un certificat SSL auto-signé et que la zone Cloudflare est configurée en mode SSL Full (Strict), qui exige un certificat valide d'une autorité de certification approuvée.
