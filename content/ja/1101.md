## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

Cloudflare エラー `1101 Worker threw a JavaScript exception` は、Cloudflare Workers スクリプトが処理されなかった JavaScript 例外をスローした場合に発生します。スクリプト内のエラーハンドリングが不十分であるか、予期しない入力によって例外が発生した可能性があります。Worker スクリプトのコードを確認し、適切な例外処理を実装してください。
