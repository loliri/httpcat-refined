## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

HTTP `523 Origin Is Unreachable` 是 [Cloudflare](https://cloudflare.com) 專用的非官方狀態碼。

當 Cloudflare 無法聯繫您的來源 Web 伺服器時，會出現 `523` 錯誤。這通常發生在 Cloudflare 與來源 Web 伺服器之間的網路裝置沒有通往來源 IP 位址的路由時。
