## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1019 Compute server error (Worker recursive call)` ocurre cuando un script de Cloudflare Worker se llama a sí mismo de forma recursiva, creando un bucle infinito. Cloudflare detecta este patrón recursivo y termina la solicitud para evitar el agotamiento de recursos.
