## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

HTTP `527 Railgun Error` 是 [Cloudflare](https://cloudflare.com) 專用的非官方狀態碼。

`527` 錯誤表示 Cloudflare 與來源伺服器的 Railgun 伺服器之間的連線中斷。Railgun 是 Cloudflare 開發的一種廣域網路（WAN）最佳化技術，用於壓縮和加速動態內容傳遞。

Cloudflare 已於 2023 年棄用並停止 Railgun 產品，因此此錯誤碼主要具有歷史意義。
