## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1034 Edge IP restricted` 表示 Cloudflare 用于连接源服务器的边缘 IP 地址受到限制或被阻止。这可能是由于源服务器上的防火墙规则阻止了 Cloudflare 的 IP 范围。
