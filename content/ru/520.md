## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

HTTP `520 Web Server Returned an Unknown Error` — неофициальный код состояния, специфичный для [Cloudflare](https://cloudflare.com).

Ошибка `520` возникает, когда исходный веб-сервер возвращает Cloudflare пустой, неизвестный или неожиданный ответ. Это часто является общей ошибкой для ситуаций, когда исходный сервер возвращает что-то, что Cloudflare не может интерпретировать.

Распространённые причины:

- Исходный сервер аварийно завершает работу или перезапускается в процессе обработки запроса.
- Исходный сервер возвращает ответ, нарушающий спецификации HTTP.
- Брандмауэр или программное обеспечение безопасности на исходном сервере блокирует запрос Cloudflare.
