## Description

Le code de statut HTTP `422 Unprocessable Content` indique que le serveur comprend le type de contenu de l'entité de requête, et que la syntaxe de l'entité de requête est correcte, mais qu'il n'a pas pu traiter les instructions contenues.

<aside class="warning"><strong>Avertissement :</strong> Le client ne doit pas répéter cette requête sans modification.</aside>
