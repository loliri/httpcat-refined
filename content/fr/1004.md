## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1004 Host not configured to serve web traffic` se produit lorsque le domaine existe dans le système Cloudflare mais n'a pas été configuré pour servir le trafic web. Cela signifie généralement que le domaine est enregistré auprès de Cloudflare mais qu'aucun hébergement web n'a été configuré.
