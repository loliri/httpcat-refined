## Descripción

El código de estado HTTP `431 Request Header Fields Too Large` indica que el servidor se niega a procesar la solicitud porque las cabeceras HTTP de la solicitud son demasiado largas. La solicitud se puede reenviar después de reducir el tamaño de las cabeceras.

Los servidores suelen producir este estado si:

- La URL `Referer` es demasiado larga
- Se envían demasiadas cookies en la solicitud
