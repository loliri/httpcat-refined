## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

HTTP `494 Request Header Too Large` es un código de estado no oficial específico de nginx. Es devuelto por el servidor cuando las cabeceras de solicitud del cliente exceden el límite establecido por la directiva `large_client_header_buffers` en la configuración de nginx.

Este error es similar al estándar `431 Request Header Fields Too Large`, pero `494` se activa en nginx durante una fase de procesamiento interno (al leer las cabeceras de la solicitud) y normalmente no contiene cuerpo de respuesta.
