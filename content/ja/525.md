## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

HTTP `525 SSL Handshake Failed` は [Cloudflare](https://cloudflare.com) 固有の非公式 HTTP ステータスコードです。

`525` エラーは、Cloudflare とオリジン Web サーバー間の SSL ハンドシェイクが失敗したことを示します。
