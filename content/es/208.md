## Descripción

El código de estado HTTP `208 Already Reported` se usa en una respuesta [207 Multi-Status](/status/207) para ahorrar espacio y evitar conflictos. Si el mismo recurso se solicita varias veces (por ejemplo, como parte de una colección) con rutas diferentes, solo el primero se informa con [200](/status/200). Las respuestas para todos los demás enlaces se informan con este código de estado 208, evitando conflictos y manteniendo la respuesta más corta.

<aside class="info"><strong>Nota:</strong> La capacidad de vincular un recurso a varias rutas es una extensión del protocolo WebDAV. Los navegadores que acceden a páginas web nunca encontrarán este código de estado.</aside>
