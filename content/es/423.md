## Descripción

El código de estado HTTP `423 Locked` indica que el recurso de destino está bloqueado, lo que significa que no se puede acceder a él. Su contenido debe contener información en formato XML de WebDAV.

<aside class="info"><strong>Nota:</strong> La capacidad de bloquear un recurso es específica de algunos servidores WebDAV. Los navegadores que acceden a páginas web nunca encontrarán este código de estado.</aside>
