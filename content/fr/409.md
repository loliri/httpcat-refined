## Description

Le code de statut HTTP `409 Conflict` indique un conflit de requête avec l'état actuel de la ressource cible.

Les conflits sont plus susceptibles de se produire en réponse à une requête `PUT`. Par exemple, vous pouvez obtenir une réponse `409` lors du téléchargement d'un fichier plus ancien que celui existant sur le serveur, entraînant un conflit de contrôle de version.
