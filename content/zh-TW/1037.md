## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

Cloudflare 錯誤 `1037 Invalid request rewrite (invalid header name)` 發生在 Cloudflare 轉換規則嘗試使用無效的標頭名稱設定或修改 HTTP 標頭時。標頭名稱必須符合 HTTP 規範。
