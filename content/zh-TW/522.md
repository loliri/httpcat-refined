## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

HTTP `522 Connection Timed Out` 是 [Cloudflare](https://cloudflare.com) 專用的非官方狀態碼。

當 Cloudflare 聯繫來源 Web 伺服器逾時時，會出現 `522` 錯誤。根據逾時發生的時間點，有兩種不同的逾時情況會導致此錯誤：

- 在建立連線之前：Cloudflare 傳送 `SYN` 後 15 秒內，來源 Web 伺服器未返回 `SYN+ACK`；
- 在建立連線之後：來源 Web 伺服器在 90 秒內未確認（`ACK`）Cloudflare 的資源請求。
