## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

Cloudflare エラー `1035 Invalid request rewrite (invalid URI path)` は、Transform Rule がリクエストを無効な URI パスに書き換えようとした場合に発生します。書き換え先のパスが正しい形式でないか、許可されていない文字が含まれている可能性があります。Transform Rule の設定を確認し、有効な URI パスを指定してください。
