## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1006 Access Denied (IP banned)` 表示访问者的 IP 地址已被 Cloudflare 自身封禁，通常是因为在 Cloudflare 网络中检测到了恶意行为。
