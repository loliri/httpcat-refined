## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1025 Plan limit exceeded` se produit lorsque le plan Cloudflare du propriétaire du site web ne prend pas en charge la fonctionnalité utilisée, ou que l'utilisation a dépassé les limites de leur plan actuel.
