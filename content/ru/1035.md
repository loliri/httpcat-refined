## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

Ошибка Cloudflare `1035 Invalid request rewrite (invalid URI path)` возникает, когда правило преобразования Cloudflare или правило перезаписи URL пытается переписать URI запроса на недопустимый путь. Переписанный путь должен быть допустимым URI-путём.
