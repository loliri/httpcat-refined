## Description

Le code de statut HTTP `510 Not Extended` est envoyé dans le contexte du framework d'extension HTTP, défini dans la RFC 2774.

Dans cette spécification, un client peut envoyer une requête contenant une déclaration d'extension décrivant l'extension à utiliser. Si le serveur reçoit une telle requête, mais que les extensions décrites ne sont pas prises en charge pour la requête, alors le serveur répond avec le code de statut `510`.
