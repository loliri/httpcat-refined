## Description

Le code de statut HTTP `201 Created` indique que la requête a réussi et a conduit à la création d'une ressource.

La nouvelle ressource, ou une description et un lien vers celle-ci, est créée avant que la réponse ne soit renvoyée, et les éléments nouvellement créés sont retournés dans le corps du message, à l'URL de la requête ou à l'URL indiquée dans la valeur de l'en-tête `Location`.

Ce code de statut est couramment utilisé comme résultat d'une requête `POST`.
