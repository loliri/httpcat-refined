## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1002 DNS points to prohibited IP` 与错误 1000 类似，表示该域名的 DNS 记录解析到了 Cloudflare 限制的 IP 地址，例如回环地址（`127.0.0.1`）或其他被禁止的地址范围。
