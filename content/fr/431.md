## Description

Le code de statut HTTP `431 Request Header Fields Too Large` indique que le serveur refuse de traiter la requête car les en-têtes HTTP de la requête sont trop longs. La requête peut être resoumise après avoir réduit la taille des en-têtes de requête.

Les serveurs produisent souvent ce statut si :

- L'URL `Referer` est trop longue
- Trop de cookies sont envoyés dans la requête
