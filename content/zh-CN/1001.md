## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1001 DNS resolution error` 表示 Cloudflare 无法解析源 Web 服务器的主机名，通常意味着源服务器的 DNS 记录不存在或配置有误。
