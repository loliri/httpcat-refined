## Descripción

El código de estado HTTP `418 I'm a teapot` indica que el servidor se niega a preparar café porque, permanentemente, es una tetera.

Una cafetera/tetera combinada que se queda temporalmente sin café debería devolver 503. Este error es una referencia al Hyper Text Coffee Pot Control Protocol definido en bromas del Día de los Inocentes (April Fools') de 1998 y 2014.

Algunos sitios web utilizan esta respuesta para solicitudes que no desean manejar, como consultas automatizadas.
