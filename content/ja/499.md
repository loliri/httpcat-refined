## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

HTTP `499` は ArcGIS（`499 Token Required`）と nginx（`499 Client Closed Request`）の両方で使用される非公式 HTTP ステータスコードです。

#### ArcGIS

`499 Token Required` を受け取った場合、サーバーは HTTP リクエストにトークンを期待していますが、送信されませんでした。

#### nginx

`499 Client Closed Request` を受け取った場合、クライアントがサーバーのタスク完了前に HTTP 接続を閉じたことを意味します。このステータスコードは nginx ログにのみ表示されます。
