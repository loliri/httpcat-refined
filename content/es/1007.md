## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1007 Access Denied (Self-signed SSL prohibited)` ocurre cuando el servidor de origen presenta un certificado SSL autofirmado y la zona de Cloudflare está configurada con el modo SSL Full (Strict), que requiere un certificado válido de una autoridad certificadora de confianza.
