## 描述

<aside class="alert"><strong>已废弃：</strong>HTTP `214 Transformation Applied` 状态码及 Warning HTTP 头已被废弃。</aside>

HTTP `214 Transformation Applied` 是一个非官方状态码，与 [Warning HTTP 头](https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Warning)配合使用，包含在 HTTP 响应消息中，用于表示已对响应内容应用了某种转换。

若代理对响应内容进行了任何转换（如修改 `content-coding`、`media-type` 或响应数据本身），且响应中尚未包含 `Warning` 头，则必须添加此状态码和 `Warning` 头。
