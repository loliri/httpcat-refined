## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

Le code de statut HTTP `523 Origin Is Unreachable` est un code non officiel spécifique à [Cloudflare](https://cloudflare.com).

L'erreur `523` se produit lorsque Cloudflare ne peut pas contacter votre serveur web d'origine. Cela se produit généralement lorsqu'un périphérique réseau entre Cloudflare et le serveur web d'origine n'a pas de route vers l'adresse IP de l'origine.
