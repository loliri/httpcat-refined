## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP `526 Invalid SSL Certificate` 是 [Cloudflare](https://cloudflare.com) 专用的非官方状态码。

当 Cloudflare 无法验证源 Web 服务器上的 SSL 证书时，会出现 `526` 错误。常见原因包括：

- 源服务器使用自签名证书，且 Cloudflare 的 SSL 模式设置为"完全（严格）"；
- 证书已过期；
- 证书的主机名与源服务器的主机名不匹配；
- 证书未被公共证书颁发机构（CA）信任。
