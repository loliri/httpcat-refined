## Description

Le code de statut HTTP `415 Unsupported Media Type` indique que le serveur refuse d'accepter la requête car le format de la charge utile n'est pas dans un format pris en charge.

Le problème de format peut être dû au `Content-Type` ou au `Content-Encoding` indiqués dans la requête, ou à la suite d'une inspection directe des données.
