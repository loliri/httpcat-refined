## Descripción

El código de respuesta HTTP `207 Multi-Status` indica que puede haber una mezcla de respuestas.

El cuerpo de la respuesta es una entidad HTTP `text/xml` o `application/xml` con un elemento raíz `multistatus`. El cuerpo XML enumera todos los códigos de respuesta individuales.

<aside class="info"><strong>Nota:</strong> La capacidad de devolver una colección de recursos es parte del protocolo WebDAV. Los navegadores que acceden a páginas web nunca encontrarán este código de estado.</aside>
