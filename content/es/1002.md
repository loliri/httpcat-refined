## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1002 DNS points to prohibited IP` es similar al error 1000 y ocurre cuando el registro DNS del dominio se resuelve en una dirección IP que Cloudflare ha restringido, como una dirección de bucle invertido (`127.0.0.1`) u otro rango prohibido.
