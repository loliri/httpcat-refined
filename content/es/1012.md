## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1012 Access denied (Cloudflare app blocked)` ocurre cuando una aplicación o integración de Cloudflare ha bloqueado la solicitud. Esto puede ocurrir cuando una aplicación de seguridad instalada en la zona de Cloudflare determina que la solicitud no debe pasar.
