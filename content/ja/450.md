## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

HTTP `450 Blocked by Windows Parental Controls` は Microsoft 固有の非公式 HTTP ステータスコードで、Windows ペアレンタルコントロールが有効になっており、リクエストされたリソースへのアクセスをブロックしていることを示します。
