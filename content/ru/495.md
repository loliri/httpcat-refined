## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

HTTP `495 SSL Certificate Error` — неофициальная ошибка клиента, специфичная для nginx, возвращаемая сервером при ошибке проверки клиентского сертификата.

- [Nginx ngx_http_ssl_module](http://nginx.org/en/docs/http/ngx_http_ssl_module.html#errors)
