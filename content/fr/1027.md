## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1027 Account suspended` se produit lorsque le compte Cloudflare associé au domaine a été suspendu, généralement en raison d'une violation des conditions d'utilisation de Cloudflare ou d'un défaut de paiement.
