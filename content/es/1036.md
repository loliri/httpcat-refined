## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1036 Invalid request rewrite (max length exceeded)` ocurre cuando una regla de transformación o de reescritura de URL de Cloudflare produce una URL reescrita que excede la longitud máxima permitida.
