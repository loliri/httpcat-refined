## Description

Le code de statut HTTP `429 Too Many Requests` indique que l'utilisateur a envoyé trop de requêtes dans un laps de temps donné (« limitation de débit »).

Un en-tête `Retry-After` peut être inclus dans cette réponse, indiquant combien de temps attendre avant de faire une nouvelle requête.
