## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1016 Origin DNS error` se produit lorsque Cloudflare ne peut pas résoudre le nom d'hôte du serveur web d'origine. Contrairement à l'erreur 1001, cette erreur indique spécifiquement un échec lors de la recherche DNS pour le serveur d'origine au moment de la requête.
