## Description

Le code de statut HTTP `300 Multiple Choices` indique que la requête a plus d'une réponse possible. L'agent utilisateur ou l'utilisateur doit en choisir une.

Comme il n'existe pas de méthode standardisée pour choisir l'une des réponses, ce code de réponse est très rarement utilisé.

Si le serveur a une préférence, il doit générer un en-tête `Location`.
