## Description

Le code de statut HTTP `506 Variant Also Negotiates` peut être donné dans le contexte de la négociation de contenu transparente (voir RFC 2295). Ce protocole permet à un client de récupérer la meilleure variante d'une ressource donnée, lorsque le serveur prend en charge plusieurs variantes.

Le code de statut Variant Also Negotiates indique une erreur de configuration interne du serveur dans laquelle la variante choisie est elle-même configurée pour participer à la négociation de contenu, et n'est donc pas un point de terminaison de négociation approprié.
