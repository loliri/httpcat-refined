## 描述

HTTP `207 Multi-Status` 响应码表示响应中可能包含多种状态的混合结果。

响应体为 `text/xml` 或 `application/xml` 格式的 HTTP 实体，根元素为 `multistatus`，XML 体中列出各个子响应的状态码。

<aside class="info"><strong>注意：</strong>返回资源集合的能力是 WebDAV 协议的一部分（可能由访问 WebDAV 服务器的 Web 应用收到）。普通浏览器访问网页时不会遇到此状态码。</aside>
