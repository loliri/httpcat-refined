## Description

Le code de statut HTTP `412 Precondition Failed` indique que l'accès à la ressource cible a été refusé. Cela se produit avec des requêtes conditionnelles sur des méthodes autres que `GET` ou `HEAD` lorsque la condition définie par les en-têtes `If-Unmodified-Since` ou `If-None-Match` n'est pas remplie.
