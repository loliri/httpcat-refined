## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

Le code de statut HTTP `419 Page Expired` est un code non officiel principalement utilisé par le framework PHP Laravel pour indiquer une incompatibilité de jeton CSRF.
