## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

Ошибка Cloudflare `1005 Access Denied (IP banned by origin)` возникает, когда исходный веб-сервер заблокировал IP-адрес посетителя. Исходный сервер активно отклоняет соединения с этого IP, и Cloudflare передаёт этот отказ.
