## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1015 You are being rate limited` ocurre cuando un visitante ha excedido el límite de tasa configurado para la zona. La función de limitación de tasa de Cloudflare permite a los propietarios de sitios web definir umbrales para el número de solicitudes permitidas desde una sola dirección IP en un período de tiempo determinado.
