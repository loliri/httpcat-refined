## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

Cloudflare エラー `1040 Invalid request rewrite (transformations disabled)` は、Transform Rules 機能がゾーンで無効になっているにもかかわらず、変換が試みられた場合に発生します。この機能を使用するには、Cloudflare ダッシュボードで Transform Rules を有効にする必要があります。現在のプランがこの機能をサポートしているかどうかも確認してください。
