## 説明

HTTP `418 I'm a teapot` クライアントエラーレスポンスコードは、サーバーが永久にティーポットであるため、コーヒーを淹れることを拒否することを示します。

コーヒーが一時的に切れているコーヒー/ティーポットの組み合わせは、代わりに `503` を返す必要があります。このエラーは 1998 年と 2014 年のエイプリルフールのジョークで定義された「Hyper Text Coffee Pot Control Protocol」への参照です。

一部の Web サイトは、自動クエリなど処理したくないリクエストに対してこのレスポンスを使用します。
