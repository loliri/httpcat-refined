## Description

Le code de statut HTTP `400 Bad Request` indique que le serveur ne peut pas ou ne veut pas traiter la requête en raison d'une erreur perçue comme étant du côté client (par exemple, une syntaxe de requête malformée, un cadrage de message de requête invalide ou un routage de requête trompeur).

<aside class="warning"><strong>Avertissement :</strong> Le client ne doit pas répéter cette requête sans modification.</aside>
