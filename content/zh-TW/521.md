## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

HTTP `521 Web Server Is Down` 是 [Cloudflare](https://cloudflare.com) 專用的非官方狀態碼。

當來源 Web 伺服器拒絕來自 Cloudflare 的連線時，會出現此錯誤。
