## Описание

<aside class="alert"><strong>Устарело:</strong> Код состояния HTTP 305 Use Proxy устарел из-за проблем безопасности, и многие HTTP-клиенты игнорируют его.</aside>

Код состояния HTTP `305 Use Proxy` — устаревший код, возвращаемый исходным сервером для указания на то, что запрошенный ресурс доступен только через прокси-сервер.

Код `305 Use Proxy` используется для информирования клиента о том, что запрошенный ресурс должен быть получен через прокси-сервер, адрес которого указывается в HTTP-заголовке `Set-Proxy` или `Location`.
