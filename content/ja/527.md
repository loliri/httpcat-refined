## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

HTTP `527 Railgun Error` は [Cloudflare](https://cloudflare.com) 固有の非公式 HTTP ステータスコードです。

`527` エラーは、Cloudflare とオリジンサーバーの Railgun サービス間の接続が中断されたことを示します。Cloudflare は 2023 年に Railgun 製品を廃止したため、このエラーコードは主に歴史的な意味を持ちます。
