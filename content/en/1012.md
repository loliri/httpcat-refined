## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1012 Access denied (Cloudflare app blocked)` occurs when a Cloudflare App or integration has blocked the request. This can happen when a security app installed on the Cloudflare zone determines the request should not be allowed through.
