## Description

Le code de statut HTTP `202 Accepted` indique que la requête a été acceptée pour traitement, mais que le traitement n'est pas terminé ; en fait, il peut ne pas avoir encore commencé.

La requête pourrait ou non finalement être traitée, car elle pourrait être refusée lorsque le traitement a réellement lieu.

`202` est non engageant, ce qui signifie qu'il n'y a aucun moyen pour HTTP d'envoyer ultérieurement une réponse asynchrone indiquant le résultat du traitement de la requête. Il est destiné aux cas où un autre processus ou serveur gère la requête, ou pour le traitement par lots.
