## Description

Le code de statut HTTP `418 I'm a teapot` indique que le serveur refuse de préparer du café car il est, de façon permanente, une théière.

Une théière/cafetière combinée qui manque temporairement de café devrait plutôt retourner 503. Cette erreur est une référence au Hyper Text Coffee Pot Control Protocol défini dans des blagues du 1er avril en 1998 et 2014.

Certains sites web utilisent cette réponse pour les requêtes qu'ils ne souhaitent pas traiter, comme les requêtes automatisées.
