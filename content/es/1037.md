## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1037 Invalid request rewrite (invalid header name)` ocurre cuando una regla de transformación de Cloudflare intenta establecer o modificar una cabecera HTTP usando un nombre de cabecera no válido. Los nombres de cabecera deben cumplir con las especificaciones HTTP.
