## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

Ошибка Cloudflare `1106 Mirage exception` возникает, когда функция Mirage Cloudflare сталкивается с ошибкой. Mirage — функция оптимизации изображений, которая загружает изображения по требованию и адаптирует их для мобильных устройств для повышения производительности на медленных соединениях.
