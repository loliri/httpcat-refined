## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1013 HTTP hostname and TLS SNI hostname mismatch` 表示 HTTP `Host` 头中提供的主机名与 TLS 服务器名称指示（SNI）扩展中的主机名不匹配。这种不匹配可能表明客户端配置有误或存在潜在的安全问题。
