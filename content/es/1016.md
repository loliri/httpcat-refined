## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1016 Origin DNS error` ocurre cuando Cloudflare no puede resolver el nombre de host del servidor web de origen. A diferencia del error 1001, este error indica específicamente un fallo durante la búsqueda DNS del servidor de origen en el momento de la solicitud.
