## Descripción

El código de estado HTTP `405 Method Not Allowed` indica que el servidor conoce el método de solicitud, pero el recurso de destino no admite este método.

El servidor debe generar un campo de cabecera `Allow` en una respuesta con código de estado `405`. El campo debe contener una lista de los métodos que el recurso de destino admite actualmente.
