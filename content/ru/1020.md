## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

Ошибка Cloudflare `1020 Access denied (Cloudflare Firewall Rule)` возникает, когда правило брандмауэра Cloudflare (теперь часть пользовательских правил WAF) заблокировало запрос. Владельцы сайтов могут настраивать правила для блокировки трафика на основе IP-адреса, страны, пользовательского агента, заголовков запроса и других критериев.
