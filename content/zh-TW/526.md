## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

HTTP `526 Invalid SSL Certificate` 是 [Cloudflare](https://cloudflare.com) 專用的非官方狀態碼。

當 Cloudflare 無法驗證來源 Web 伺服器上的 SSL 憑證時，會出現 `526` 錯誤。通常發生在以下情況：

- 來源伺服器上的 SSL 憑證是自簽憑證，且 Cloudflare 的 SSL 模式設定為「完整（嚴格）」；
- 憑證已過期；
- 憑證的主機名稱與來源的主機名稱不符；
- 憑證不受公共憑證授權機構信任。
