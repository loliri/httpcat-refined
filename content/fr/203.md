## Description

Le code de statut HTTP `203 Non-Authoritative Information` indique que la requête a réussi, mais que la charge utile incluse a été modifiée par un proxy de transformation par rapport à la réponse `200 OK` du serveur d'origine.

La réponse `203` est similaire à la valeur `214` (`Transformation Applied`) du code d'en-tête `Warning`, qui a l'avantage supplémentaire d'être applicable aux réponses avec n'importe quel code de statut.
