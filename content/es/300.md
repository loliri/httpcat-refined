## Descripción

El código de estado HTTP `300 Multiple Choices` indica que la solicitud tiene más de una respuesta posible. El agente de usuario o el usuario debe elegir una de ellas.

Como no existe una forma estandarizada de elegir una de las respuestas, este código de respuesta se usa muy raramente.

Si el servidor tiene una opción preferida, debe generar una cabecera `Location`.
