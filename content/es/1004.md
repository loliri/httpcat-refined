## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1004 Host not configured to serve web traffic` ocurre cuando el dominio existe en el sistema de Cloudflare pero no se ha configurado para servir tráfico web. Esto suele significar que el dominio está registrado en Cloudflare pero no se ha configurado el alojamiento web.
