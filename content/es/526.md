## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

HTTP `526 Invalid SSL Certificate` es un código de estado no oficial específico de [Cloudflare](https://cloudflare.com).

El error `526` ocurre cuando Cloudflare no puede validar el certificado SSL en el servidor web de origen. Suele ocurrir en los siguientes casos:

- El certificado SSL en el origen es autofirmado y el modo SSL de Cloudflare está configurado como Full (Strict).
- El certificado ha expirado.
- El nombre de host del certificado no coincide con el nombre de host del origen.
- El certificado no es de confianza para una autoridad certificadora pública.
