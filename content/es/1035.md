## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1035 Invalid request rewrite (invalid URI path)` ocurre cuando una regla de transformación o una regla de reescritura de URL de Cloudflare intenta reescribir el URI de la solicitud a una ruta no válida. La ruta reescrita debe ser una ruta URI válida.
