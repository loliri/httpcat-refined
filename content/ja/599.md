## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

HTTP `599 Network Connect Timeout Error` は、プロキシの背後でネットワーク接続タイムアウトが発生したことをプロキシの前のクライアントに通知するために一部のプロキシが使用する非公式 HTTP ステータスコードです。
