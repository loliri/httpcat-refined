## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1011 Access denied (Hotlink protection)` se produit lorsqu'une requête est bloquée par la fonctionnalité de protection contre le hotlinking de Cloudflare. La protection contre le hotlinking empêche d'autres sites web de créer des liens directs vers les images ou autres ressources de votre site, ce qui consommerait votre bande passante sans votre permission.
