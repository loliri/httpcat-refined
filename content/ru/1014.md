## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

Ошибка Cloudflare `1014 CNAME cross-user banned` возникает, когда CNAME-запись в одной учётной записи Cloudflare указывает на домен в другой учётной записи Cloudflare, и эта межаккаунтная CNAME заблокирована. Cloudflare ограничивает определённые конфигурации CNAME между пользователями для предотвращения злоупотреблений.
