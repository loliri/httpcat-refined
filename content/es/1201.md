## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1201 Origin returned malformed HTTP` ocurre cuando el servidor web de origen devuelve una respuesta HTTP que no se ajusta a la especificación HTTP. Cloudflare no puede analizar ni transmitir una respuesta mal formada al cliente.
