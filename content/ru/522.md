## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

HTTP `522 Connection Timed Out` — неофициальный код состояния, специфичный для [Cloudflare](https://cloudflare.com).

Ошибка `522` возникает, когда Cloudflare превышает время ожидания при обращении к исходному веб-серверу. Два разных тайм-аута вызывают эту ошибку:

- До установления соединения: исходный веб-сервер не возвращает `SYN+ACK` в течение 15 секунд после отправки Cloudflare `SYN`.
- После установления соединения: исходный веб-сервер не подтверждает (`ACK`) запрос ресурса Cloudflare в течение 90 секунд.
