## Описание

<aside class="alert"><strong>Устарело:</strong> Код состояния HTTP 214 Transformation Applied и заголовок HTTP Warning устарели.</aside>

HTTP `214 Transformation Applied` — неофициальный код состояния, используемый совместно с [заголовком HTTP Warning](https://developer.mozilla.org/ru/docs/Web/HTTP/Headers/Warning) и включаемый в HTTP-ответы для указания на то, что к представлению было применено преобразование.

Этот код состояния и заголовок `Warning` ДОЛЖНЫ добавляться прокси, если он применяет любое преобразование к представлению, если только заголовок `Warning` уже не присутствует в ответе.
