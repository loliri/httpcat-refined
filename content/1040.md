## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1040 Invalid request rewrite (transformations disabled)` occurs when a request rewrite is attempted but the Transform Rules feature has been disabled for the zone.
