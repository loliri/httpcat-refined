## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

Cloudflare エラー `1037 Invalid request rewrite (invalid header name)` は、Transform Rule が無効なヘッダー名を使用しようとした場合に発生します。HTTP ヘッダー名には使用できない文字があり、それらを含む名前は拒否されます。Transform Rule の設定を確認し、有効な HTTP ヘッダー名を使用してください。
