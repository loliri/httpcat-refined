## Description

Le code de statut HTTP `208 Already Reported` est utilisé dans une réponse [207 Multi-Status](/status/207) pour économiser de l'espace et éviter les conflits. Si la même ressource est demandée plusieurs fois (par exemple dans le cadre d'une collection), avec des chemins différents, seule la première est signalée avec [200](/status/200). Les réponses pour toutes les autres liaisons seront signalées avec ce code de statut 208, afin qu'aucun conflit ne soit créé et que la réponse reste plus courte.

<aside class="info"><strong>Note :</strong> La capacité à lier une ressource à plusieurs chemins est une extension du protocole WebDAV. Les navigateurs accédant à des pages web ne rencontreront jamais ce code de statut.</aside>
