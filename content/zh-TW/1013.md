## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

Cloudflare 錯誤 `1013 HTTP hostname and TLS SNI hostname mismatch` 發生在 HTTP `Host` 標頭中提供的主機名稱與 TLS 伺服器名稱指示（SNI）擴充中的主機名稱不符時。此不符情況可能表示客戶端設定錯誤或存在潛在的安全問題。
