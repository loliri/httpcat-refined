## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1005 Access Denied (IP banned by origin)` se produit lorsque le serveur web d'origine a banni l'adresse IP du visiteur. Le serveur d'origine refuse activement les connexions de cette IP, et Cloudflare relaie ce refus.
