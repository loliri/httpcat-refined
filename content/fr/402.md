## Description

<aside class="warning"><strong>Expérimental :</strong> Il s'agit d'une technologie expérimentale. Vérifiez attentivement le tableau de compatibilité des navigateurs avant de l'utiliser en production.</aside>

Le code de statut HTTP `402 Payment Required` est un code de statut non standard réservé pour une utilisation future. Ce code de statut a été créé pour permettre des systèmes de paiement numérique ou de micropaiement et indiquerait que le contenu demandé n'est pas disponible tant que le client n'effectue pas un paiement.
