## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1012 Access denied (Cloudflare app blocked)` se produit lorsqu'une application ou intégration Cloudflare a bloqué la requête. Cela peut se produire lorsqu'une application de sécurité installée sur la zone Cloudflare détermine que la requête ne doit pas être autorisée à passer.
