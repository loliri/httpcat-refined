## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1006 Access Denied (IP banned)` occurs when the visitor's IP address has been banned by Cloudflare itself, typically due to malicious activity detected across the Cloudflare network.
