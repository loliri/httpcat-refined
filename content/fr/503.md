## Description

Le code de statut HTTP `503 Service Unavailable` indique que le serveur n'est pas prêt à gérer la requête.

Les causes courantes sont un serveur en panne pour maintenance ou surchargé. Cette réponse doit être utilisée pour des conditions temporaires et l'en-tête HTTP `Retry-After` doit, si possible, contenir le temps estimé de récupération du service.
