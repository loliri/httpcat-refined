## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1002 DNS points to prohibited IP` is similar to error 1000 and occurs when the DNS record for the domain resolves to an IP address that Cloudflare has restricted, such as a loopback address (`127.0.0.1`) or another prohibited range.
