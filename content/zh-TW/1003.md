## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

Cloudflare 錯誤 `1003 Direct IP access not allowed` 發生在訪客嘗試直接存取 Cloudflare IP 位址而非透過網域名稱存取時。Cloudflare 不允許直接存取其基礎設施的 IP 位址。
