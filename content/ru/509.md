## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

HTTP `509 Bandwidth Limit Exceeded` — неофициальный код состояния, специфичный для веб-сервера Apache и cPanel, указывающий, что клиент веб-хостинга превысил выделенный лимит трафика на стороне сервера.
