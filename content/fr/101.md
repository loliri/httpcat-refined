## Description

Le code de réponse HTTP `101 Switching Protocols` indique le protocole vers lequel le serveur bascule, tel que spécifié dans l'en-tête de requête [Upgrade](https://developer.mozilla.org/fr/docs/Web/HTTP/Headers/Upgrade) reçu du client.

Le serveur inclut dans cette réponse un en-tête de réponse [Upgrade](https://developer.mozilla.org/fr/docs/Web/HTTP/Headers/Upgrade) pour indiquer le protocole vers lequel il a basculé.
