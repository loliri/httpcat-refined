## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

Le code de statut HTTP `522 Connection Timed Out` est un code non officiel spécifique à [Cloudflare](https://cloudflare.com).

L'erreur `522` se produit lorsque Cloudflare expire en contactant le serveur web d'origine. Deux délais d'expiration différents provoquent cette erreur :

- Avant l'établissement d'une connexion : le serveur web d'origine ne renvoie pas de `SYN+ACK` à Cloudflare dans les 15 secondes suivant l'envoi d'un `SYN` par Cloudflare.
- Après l'établissement d'une connexion : le serveur web d'origine n'accuse pas réception (`ACK`) de la requête de ressource de Cloudflare dans les 90 secondes.
