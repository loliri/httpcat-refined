## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

Cloudflare 錯誤 `1201 Origin returned malformed HTTP` 發生在來源 Web 伺服器返回不符合 HTTP 規範的回應時。Cloudflare 無法解析或轉發格式錯誤的回應給客戶端。
