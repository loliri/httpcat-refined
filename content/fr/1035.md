## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1035 Invalid request rewrite (invalid URI path)` se produit lorsqu'une règle de transformation Cloudflare ou une règle de réécriture d'URL tente de réécrire l'URI de la requête vers un chemin invalide. Le chemin réécrit doit être un chemin URI valide.
