## Description

<aside class="alert"><strong>Obsolète :</strong> Le code de statut HTTP 305 Use Proxy a été déprécié en raison de problèmes de sécurité et de nombreux clients HTTP ignorent ce code de statut.</aside>

Le code de statut HTTP `305 Use Proxy` est un code de statut déprécié renvoyé par un serveur d'origine pour indiquer que la ressource demandée n'est accessible que via un serveur proxy.

Le code de statut `305 Use Proxy` est utilisé pour informer le client que la ressource demandée doit être obtenue via un serveur proxy et inclut l'adresse de ce proxy dans l'en-tête HTTP `Set-Proxy` ou `Location`.
