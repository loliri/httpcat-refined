## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

HTTP `494 Request Header Too Large` — неофициальный код состояния, специфичный для nginx. Возвращается сервером, когда заголовки запроса клиента превышают лимит, установленный директивой `large_client_header_buffers` в конфигурации nginx.

Эта ошибка аналогична стандартному `431 Request Header Fields Too Large`, но `494` срабатывает в nginx на этапе внутренней обработки (при чтении заголовков запроса) и обычно не содержит тела ответа.
