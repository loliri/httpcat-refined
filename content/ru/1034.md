## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

Ошибка Cloudflare `1034 Edge IP restricted` возникает, когда граничный IP-адрес, используемый Cloudflare для подключения к источнику, ограничен или заблокирован. Это может произойти из-за правил брандмауэра на исходном сервере, блокирующих диапазоны IP-адресов Cloudflare.
