## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1020 Access denied (Cloudflare Firewall Rule)` ocurre cuando una regla de firewall de Cloudflare (ahora parte de las reglas personalizadas de WAF) ha bloqueado la solicitud. Los propietarios de sitios web pueden configurar reglas para bloquear el tráfico según la dirección IP, el país, el agente de usuario, las cabeceras de solicitud y otros criterios.
