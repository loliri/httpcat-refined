## Description

Le code de statut HTTP `511 Network Authentication Required` indique que le client doit s'authentifier pour obtenir l'accès au réseau.

Ce statut n'est pas généré par les serveurs d'origine, mais par les proxys interceptifs qui contrôlent l'accès au réseau.

Les opérateurs de réseau exigent parfois une authentification, l'acceptation de conditions ou d'autres interactions utilisateur avant d'accorder l'accès (par exemple dans un cybercafé ou à un aéroport).
