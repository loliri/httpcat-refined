## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1005 Access Denied (IP banned by origin)` ocurre cuando el servidor web de origen ha bloqueado la dirección IP del visitante. El servidor de origen rechaza activamente las conexiones desde esa IP, y Cloudflare transmite ese rechazo.
