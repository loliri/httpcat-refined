## Description

Le code de statut HTTP `424 Failed Dependency` indique que la méthode n'a pas pu être effectuée sur la ressource car l'action demandée dépendait d'une autre action, et cette action a échoué.

Les serveurs web ordinaires ne retournent normalement pas ce code de statut. Mais d'autres protocoles, comme WebDAV, peuvent le retourner.
