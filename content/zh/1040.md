## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1040 Invalid request rewrite (transformations disabled)` 表示尝试进行请求重写，但该区域的转换规则功能已被禁用。
