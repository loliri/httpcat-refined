## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1102 Worker exceeded CPU time limit` se produit lorsqu'un script Cloudflare Worker dépasse la limite de temps CPU allouée pour son exécution. Cloudflare Workers a des limites strictes de temps CPU pour assurer une utilisation équitable des ressources entre tous les clients.
