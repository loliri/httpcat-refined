## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

HTTP `520 Web Server Returned an Unknown Error` は [Cloudflare](https://cloudflare.com) 固有の非公式 HTTP ステータスコードです。

`520` エラーは、オリジン Web サーバーが Cloudflare に空、不明、または予期しないレスポンスを返した場合に発生します。
