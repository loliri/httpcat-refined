## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1037 Invalid request rewrite (invalid header name)` occurs when a Cloudflare Transform Rule attempts to set or modify an HTTP header using an invalid header name. Header names must conform to HTTP specifications.
