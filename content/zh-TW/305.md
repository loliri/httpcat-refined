## 描述

<aside class="alert"><strong>已棄用：</strong>此狀態碼已被棄用，許多 HTTP 客戶端會忽略它。</aside>

HTTP `305 Use Proxy` 是已棄用的狀態碼，由來源伺服器返回，表示所請求的資源只能透過代理伺服器存取。此狀態碼因安全疑慮已被棄用。

`305 Use Proxy` 狀態碼用於告知客戶端必須透過代理伺服器取得所請求的資源，並在 `Set-Proxy` 或 `Location` HTTP 標頭中包含代理位址。
