## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1016 Origin DNS error` 表示 Cloudflare 无法解析源 Web 服务器的主机名。与错误 1001 不同，此错误特指在请求发生时对源服务器进行 DNS 查询失败。
