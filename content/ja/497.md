## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

HTTP `497 HTTP Request Sent to HTTPS Port` は nginx 固有の非公式 HTTP ステータスコードで、通常の HTTP リクエストが HTTPS ポートに送信されたことを示します。
