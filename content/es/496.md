## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

HTTP `496 SSL Certificate Required` es un error de cliente no oficial específico de nginx, devuelto por el servidor cuando el cliente no ha presentado el certificado requerido.
