## 説明

HTTP `423 Locked` エラーレスポンスコードは、ターゲットリソースがロックされており、アクセスできないことを示します。コンテンツには WebDAV の XML 形式の情報が含まれている必要があります。

<aside class="info"><strong>注意：</strong>リソースをロックする機能は一部の WebDAV サーバーに固有のものです。ブラウザで Web ページにアクセスする際にこのステータスコードが表示されることはありません。</aside>
