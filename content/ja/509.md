## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

HTTP `509 Bandwidth Limit Exceeded` は Apache Web サーバーと cPanel 固有の非公式 HTTP ステータスコードで、Web ホスティングクライアントがサーバー側の帯域幅割り当てを超えたことを示します。
