## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1101 Worker threw a JavaScript exception` se produit lorsqu'un script Cloudflare Worker lance une exception JavaScript non gérée pendant l'exécution. Le Worker n'a pas pu traiter la requête en raison d'une erreur d'exécution dans le script.
