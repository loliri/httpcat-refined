## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

Cloudflare エラー `1105 Rocket Loader exception` は、Cloudflare の Rocket Loader 機能がページの JavaScript を処理する際にエラーが発生した場合に起こります。Rocket Loader はページの読み込みを高速化するために JavaScript の非同期読み込みを行いますが、特定のスクリプトとの互換性の問題が生じることがあります。Cloudflare ダッシュボードで Rocket Loader を無効にして問題が解決するか確認してください。
