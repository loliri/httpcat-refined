## 描述

HTTP `431 Request Header Fields Too Large` 回應狀態碼表示伺服器拒絕處理請求，因為請求的 HTTP 標頭過長。可以在縮減請求標頭大小後重新提交請求。

`431` 可用於請求標頭總大小過大，或單個標頭欄位過大的情況。為幫助遇到此錯誤的使用者，應在回應主體中說明是哪種情況，最好也指出哪些標頭過大，讓使用者嘗試修復問題（例如清除 Cookie）。

伺服器通常在以下情況產生此狀態：

- `Referer` URL 過長
- 請求中傳送了過多 Cookie
