## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

HTTP `599 Network Connect Timeout Error` — неофициальный код состояния, используемый некоторыми прокси для сигнализации клиенту перед прокси о тайм-ауте сетевого соединения за прокси.
