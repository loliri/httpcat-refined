## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

Ошибка Cloudflare `1101 Worker threw a JavaScript exception` возникает, когда скрипт Cloudflare Worker генерирует необработанное исключение JavaScript во время выполнения. Worker не смог обработать запрос из-за ошибки времени выполнения в скрипте.
