## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1008 Access Denied (ASN banned)` se produit lorsque le numéro de système autonome (ASN) associé à l'adresse IP du visiteur a été banni. Un ASN représente un ensemble de préfixes IP gérés par une seule organisation, telle qu'un FAI ou un fournisseur d'hébergement.
