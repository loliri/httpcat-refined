## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

HTTP `525 SSL Handshake Failed` — неофициальный код состояния, специфичный для [Cloudflare](https://cloudflare.com).

Ошибка `525` указывает, что SSL-рукопожатие между Cloudflare и исходным веб-сервером завершилось неудачей. Ошибка `525` возникает при одновременном выполнении двух условий:

- SSL-рукопожатие между Cloudflare и исходным веб-сервером завершается неудачей.
- В настройках Cloudflare SSL/TLS установлен режим Full или Full (Strict).
