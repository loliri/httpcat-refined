## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

HTTP `521 Web Server Is Down` es un código de estado no oficial específico de [Cloudflare](https://cloudflare.com).

Este error ocurre cuando el servidor web de origen rechaza las conexiones de Cloudflare.
