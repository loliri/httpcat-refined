## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

Ошибка Cloudflare `1008 Access Denied (ASN banned)` возникает, когда номер автономной системы (ASN), связанный с IP-адресом посетителя, заблокирован. ASN представляет собой набор IP-префиксов, управляемых одной организацией, например интернет-провайдером или хостинг-провайдером.
