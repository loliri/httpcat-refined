## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

HTTP `497 HTTP Request Sent to HTTPS Port` 是 nginx 專用的非官方狀態碼，由伺服器返回以表示普通 HTTP 請求被傳送至 HTTPS 連接埠。

收到 `497 HTTP Request Sent to HTTPS Port` 狀態碼時，HTTP 請求本身可能是有效的，但伺服器拒絕處理，因為它被傳送至 HTTPS 連接埠。此錯誤與 `400 Bad Request` 相關。
