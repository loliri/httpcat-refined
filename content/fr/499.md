## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

Le code de statut HTTP `499` est un code non officiel spécifique à la fois à ArcGIS avec `499 Token Required` et à nginx avec `499 Client Closed Request`.

#### ArcGIS

Lorsque le code `499 Token Required` est reçu, le serveur attend un jeton dans la requête HTTP mais il n'a pas été soumis.

#### nginx

Lorsque le code `499 Client Closed Request` est reçu, cela signifie que le client a fermé la connexion HTTP avant que le serveur ne termine la tâche. La réponse finale ne peut donc pas être envoyée et ce code de statut n'est disponible que dans les journaux nginx.
