## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

Le code de statut HTTP `599 Network Connect Timeout Error` est un code non officiel utilisé par certains proxys pour signaler un délai de connexion réseau derrière le proxy à un client devant le proxy.
