## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

Cloudflare エラー `1002 DNS points to prohibited IP` は、ドメインの DNS レコードがループバックアドレス（`127.0.0.1` など）やその他の制限された IP アドレスに解決される場合に発生します。このような IP アドレスへのルーティングは許可されていないため、Cloudflare はリクエストを処理できません。DNS レコードを有効なオリジンサーバーの IP アドレスに変更してください。
