## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1101 Worker threw a JavaScript exception` 表示某个 Cloudflare Worker 脚本在执行过程中抛出了未处理的 JavaScript 异常。由于脚本中存在运行时错误，Worker 未能处理该请求。
