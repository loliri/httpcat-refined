## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

HTTP `450 Blocked by Windows Parental Controls` es un código de estado no oficial específico de Microsoft, devuelto por el servidor para indicar que el control parental está activado y bloquea el acceso al recurso solicitado.
