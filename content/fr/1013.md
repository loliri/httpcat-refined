## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1013 HTTP hostname and TLS SNI hostname mismatch` se produit lorsque le nom d'hôte fourni dans l'en-tête HTTP `Host` ne correspond pas au nom d'hôte de l'extension TLS Server Name Indication (SNI). Cette discordance peut indiquer un client mal configuré ou un problème de sécurité potentiel.
