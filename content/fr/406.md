## Description

Le code de statut HTTP `406 Not Acceptable` indique que le serveur ne peut pas produire une réponse correspondant à la liste des valeurs acceptables définies dans les en-têtes de négociation de contenu proactive de la requête, et que le serveur n'est pas disposé à fournir une représentation par défaut.

Les en-têtes de négociation de contenu proactive incluent :

- `Accept`
- `Accept-Encoding`
- `Accept-Language`

En pratique, cette erreur est très rarement utilisée. Au lieu de répondre avec ce code d'erreur, les serveurs ignorent l'en-tête concerné et servent une page réelle à l'utilisateur.
