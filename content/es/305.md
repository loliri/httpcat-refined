## Descripción

<aside class="alert"><strong>Obsoleto:</strong> El código de estado HTTP 305 Use Proxy ha quedado obsoleto debido a problemas de seguridad y muchos clientes HTTP lo ignoran.</aside>

El código de estado HTTP `305 Use Proxy` es un código obsoleto devuelto por un servidor de origen para indicar que el recurso solicitado solo puede accederse a través de un servidor proxy.

El código `305 Use Proxy` se utiliza para informar al cliente de que el recurso solicitado debe obtenerse a través de un servidor proxy e incluye la dirección de ese proxy en la cabecera HTTP `Set-Proxy` o `Location`.
