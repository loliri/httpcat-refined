## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1008 Access Denied (ASN banned)` ocurre cuando el número de sistema autónomo (ASN) asociado con la dirección IP del visitante ha sido bloqueado. Un ASN representa un conjunto de prefijos IP gestionados por una sola organización, como un ISP o un proveedor de alojamiento.
