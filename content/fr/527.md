## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

Le code de statut HTTP `527 Railgun Error` est un code non officiel spécifique à [Cloudflare](https://cloudflare.com).

L'erreur `527` indique une connexion interrompue entre Cloudflare et le serveur Railgun du serveur d'origine. Railgun était une technologie d'optimisation WAN développée par Cloudflare qui compressait et accélérait la livraison de contenu dynamique.

Cloudflare a déprécié et abandonné le produit Railgun en 2023, ce code d'erreur est donc largement historique.
