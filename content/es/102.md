## Descripción

<aside class="alert"><strong>Obsoleto:</strong> Este código de estado está obsoleto. Cuando se utiliza, los clientes pueden seguir aceptándolo, pero simplemente lo ignorarán.</aside>

El código de estado informativo HTTP `102 Processing` indica al cliente que el servidor ha recibido la solicitud completa y la está procesando.

Este código de estado solo se envía si el servidor espera que la solicitud tarde un tiempo significativo. Le indica al cliente que su solicitud no está perdida.
