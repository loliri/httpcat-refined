## 説明

HTTP `500 Internal Server Error` サーバーエラーレスポンスコードは、サーバーがリクエストを処理できなかった予期しない状況に遭遇したことを示します。

このエラーレスポンスは汎用的な「キャッチオール」レスポンスです。通常、サーバーはより適切な `5xx` エラーコードを見つけられないことを示します。
