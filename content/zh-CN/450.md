## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP `450 Blocked by Windows Parental Controls` 是微软专用的非官方状态码，由服务器返回，表示 Windows 家长控制功能已开启并阻止了对所请求资源的访问。

收到 `450 Blocked by Windows Parental Controls` 状态码时，表示 Windows 家长控制已启用，并拒绝访问目标资源。
