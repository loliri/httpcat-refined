## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

HTTP `527 Railgun Error` es un código de estado no oficial específico de [Cloudflare](https://cloudflare.com).

El error `527` indica una conexión interrumpida entre Cloudflare y el servidor Railgun del servidor de origen. Railgun era una tecnología de optimización WAN desarrollada por Cloudflare que comprimía y aceleraba la entrega de contenido dinámico.

Cloudflare descontinuó el producto Railgun en 2023, por lo que este código de error es en gran medida histórico.
