## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP `527 Railgun Error` 是 [Cloudflare](https://cloudflare.com) 专用的非官方状态码。

`527` 错误表示 Cloudflare 与源服务器的 Railgun 服务之间的连接中断。Railgun 是 Cloudflare 开发的一种广域网优化技术，通过压缩和加速动态内容的传输来提升性能。

Cloudflare 已于 2023 年废弃并停止了 Railgun 产品，因此此错误码目前主要具有历史意义。
