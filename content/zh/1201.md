## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1201 Origin returned malformed HTTP` 表示源 Web 服务器返回了不符合 HTTP 规范的响应。Cloudflare 无法解析或转发格式错误的响应给客户端。
