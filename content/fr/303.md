## Description

Le code de statut HTTP `303 See Other` indique que les redirections ne pointent pas vers la ressource demandée elle-même, mais vers une autre page (telle qu'une page de confirmation, une représentation d'un objet du monde réel, ou une page de progression d'un téléchargement).

Ce code de réponse est souvent renvoyé comme résultat d'une requête `PUT` ou `POST`. La méthode utilisée pour afficher cette page redirigée est toujours `GET`.
