## 描述

HTTP `207 Multi-Status` 回應碼表示可能包含多種回應的混合結果。回應主體是帶有 `multistatus` 根元素的 `text/xml` 或 `application/xml` HTTP 實體，XML 主體會列出所有個別回應碼。

<aside class="info"><strong>注意：</strong>返回資源集合的能力是 WebDAV 協定的一部分（可能由存取 WebDAV 伺服器的 Web 應用程式收到）。瀏覽器存取網頁時不會遇到此狀態碼。</aside>
