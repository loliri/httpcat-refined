## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

Le code de statut HTTP `520 Web Server Returned an Unknown Error` est un code non officiel spécifique à [Cloudflare](https://cloudflare.com).

L'erreur `520` se produit lorsque le serveur web d'origine renvoie une réponse vide, inconnue ou inattendue à Cloudflare. C'est souvent une erreur fourre-tout pour les situations où le serveur d'origine renvoie quelque chose que Cloudflare ne peut pas interpréter.

Les causes courantes incluent :

- Le serveur d'origine plante ou redémarre en pleine requête.
- Le serveur d'origine renvoie une réponse qui viole les spécifications HTTP.
- Le pare-feu ou le logiciel de sécurité sur l'origine bloque la requête de Cloudflare.
