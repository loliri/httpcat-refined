## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

Cloudflare 錯誤 `1001 DNS resolution error` 發生在 Cloudflare 無法解析來源 Web 伺服器的主機名稱時。這通常表示來源的 DNS 記錄不存在或設定錯誤。
