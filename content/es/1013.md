## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1013 HTTP hostname and TLS SNI hostname mismatch` ocurre cuando el nombre de host proporcionado en la cabecera HTTP `Host` no coincide con el nombre de host en la extensión TLS Server Name Indication (SNI). Esta discrepancia puede indicar un cliente mal configurado o un posible problema de seguridad.
