## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

HTTP `522 Connection Timed Out` es un código de estado no oficial específico de [Cloudflare](https://cloudflare.com).

El error `522` ocurre cuando Cloudflare agota el tiempo al contactar con el servidor web de origen. Dos tiempos de espera diferentes provocan este error:

- Antes de establecer una conexión: el servidor web de origen no devuelve un `SYN+ACK` a Cloudflare en los 15 segundos posteriores al envío de un `SYN` por parte de Cloudflare.
- Después de establecer una conexión: el servidor web de origen no confirma (`ACK`) la solicitud de recurso de Cloudflare en 90 segundos.
