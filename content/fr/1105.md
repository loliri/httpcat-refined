## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1105 Rocket Loader exception` se produit lorsque la fonctionnalité Rocket Loader de Cloudflare rencontre une erreur lors du traitement du JavaScript sur la page. Rocket Loader est une fonctionnalité qui charge le JavaScript de manière asynchrone pour améliorer les performances de chargement des pages.
