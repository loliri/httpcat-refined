## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

Le code de statut HTTP `444 No Response` est un code non officiel spécifique à nginx, indiquant que la connexion HTTP a été fermée. Aucune information, pas même ce code de statut, n'est envoyée au client.

Lorsque le code `444 No Response` est généré, le serveur ne renvoie aucune information au client et ferme la connexion HTTP. Ce message d'erreur se trouve dans les journaux nginx et n'est pas envoyé au client. Il est utile pour traiter les requêtes HTTP malveillantes, telles que celles incluant un en-tête `Host` illégal.
