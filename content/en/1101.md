## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1101 Worker threw a JavaScript exception` occurs when a Cloudflare Worker script throws an unhandled JavaScript exception during execution. The Worker failed to handle the request due to a runtime error in the script.
