## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1023 Host not configured (Page Rules)` 表示该域名在 Cloudflare 页面规则系统中未正确配置。当某条页面规则引用了在 Cloudflare 区域中未激活或未正确设置的主机名时，可能会出现此错误。
