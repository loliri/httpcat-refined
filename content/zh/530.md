## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP `530 Site Frozen` 是 Cloudflare 和 Pantheon 专用的非官方状态码。

#### Cloudflare

Cloudflare 在用户与网站服务器之间发生问题时使用 `530` 状态码，并附带更详细的 `1XXX` HTTP 状态码，具体错误取决于实际情况。

#### Pantheon

当非活跃的沙盒站点拒绝 HTTP 请求时，Pantheon 会返回 `530 Site Frozen`。可通过在 Pantheon 用户控制台中重新激活该站点来解决此问题。
