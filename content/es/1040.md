## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1040 Invalid request rewrite (transformations disabled)` ocurre cuando se intenta una reescritura de solicitud pero la función de reglas de transformación está desactivada para la zona.
