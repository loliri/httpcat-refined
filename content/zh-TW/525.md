## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

HTTP `525 SSL Handshake Failed` 是 [Cloudflare](https://cloudflare.com) 專用的非官方狀態碼。

`525` 錯誤表示 Cloudflare 與來源 Web 伺服器之間的 SSL 握手失敗。當以下兩個條件同時成立時，會出現 `525` 錯誤：

- Cloudflare 與來源 Web 伺服器之間的 SSL 握手失敗；
- 在 Cloudflare SSL/TLS 應用程式的概覽標籤中設定了「完整」或「完整（嚴格）」SSL 模式。

<aside class="info"><strong>注意：</strong>若您的主機供應商頻繁更改來源 Web 伺服器的 IP 位址，請參閱 Cloudflare 關於動態 DNS 更新的文件。</aside>
