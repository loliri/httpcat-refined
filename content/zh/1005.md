## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1005 Access Denied (IP banned by origin)` 表示源 Web 服务器已封禁访问者的 IP 地址，源服务器主动拒绝来自该 IP 的连接，Cloudflare 将此拒绝转达给客户端。
