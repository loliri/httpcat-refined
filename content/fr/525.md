## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

Le code de statut HTTP `525 SSL Handshake Failed` est un code non officiel spécifique à [Cloudflare](https://cloudflare.com).

L'erreur `525` indique que la poignée de main SSL entre Cloudflare et le serveur web d'origine a échoué. L'erreur `525` se produit lorsque ces deux conditions sont remplies :

- La poignée de main SSL échoue entre Cloudflare et le serveur web d'origine, et
- Le mode SSL Full ou Full (Strict) est défini dans l'onglet Vue d'ensemble de votre application Cloudflare SSL/TLS.
