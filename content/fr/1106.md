## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1106 Mirage exception` se produit lorsque la fonctionnalité Mirage de Cloudflare rencontre une erreur. Mirage est une fonctionnalité d'optimisation d'images qui charge les images à la demande et les adapte aux appareils mobiles pour améliorer les performances sur les connexions plus lentes.
