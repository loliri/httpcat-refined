## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1041 Invalid request rewrite (header modification rejected)` ocurre cuando una regla de transformación de Cloudflare intenta modificar una cabecera HTTP protegida o de solo lectura que no se puede cambiar.
