## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

Ошибка Cloudflare `1041 Invalid request rewrite (header modification rejected)` возникает, когда правило преобразования Cloudflare пытается изменить защищённый или доступный только для чтения HTTP-заголовок, который не может быть изменён.
