## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

HTTP `530 Site Frozen` 是 Cloudflare 和 Pantheon 專用的非官方狀態碼。

#### Cloudflare

Cloudflare 在使用者與網站伺服器之間發生問題時使用 `530` 狀態碼，並附帶更詳細的 HTTP 狀態碼 `1XXX`，具體錯誤取決於特定情況。

#### Pantheon

當非活躍的沙盒網站拒絕 HTTP 請求時，Pantheon 返回 `530 Site Frozen` 狀態碼。可透過在 Pantheon 使用者儀表板中重新啟用網站來解決此問題。
