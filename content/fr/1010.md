## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1010 Access denied: Browser threat score too high` se produit lorsque l'intelligence des menaces de Cloudflare détermine que le navigateur ou l'adresse IP du visiteur a un score de menace élevé, indiquant une activité potentiellement malveillante telle que du spam, du scraping ou un comportement de bot.
