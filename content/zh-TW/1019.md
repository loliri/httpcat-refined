## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

Cloudflare 錯誤 `1019 Compute server error (Worker recursive call)` 發生在 Cloudflare Worker 腳本遞迴呼叫自身，造成無限迴圈時。Cloudflare 偵測到此遞迴模式後會終止請求，以防止資源耗盡。
