## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1007 Access Denied (Self-signed SSL prohibited)` occurs when the origin server presents a self-signed SSL certificate and the Cloudflare zone is configured with Full (Strict) SSL mode, which requires a valid certificate from a trusted Certificate Authority.
