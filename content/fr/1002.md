## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1002 DNS points to prohibited IP` est similaire à l'erreur 1000 et se produit lorsque l'enregistrement DNS du domaine se résout en une adresse IP que Cloudflare a restreinte, telle qu'une adresse de bouclage (`127.0.0.1`) ou une autre plage interdite.
