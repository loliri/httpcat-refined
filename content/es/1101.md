## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1101 Worker threw a JavaScript exception` ocurre cuando un script de Cloudflare Worker lanza una excepción de JavaScript no controlada durante la ejecución. El Worker no pudo manejar la solicitud debido a un error de tiempo de ejecución en el script.
