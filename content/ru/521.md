## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

HTTP `521 Web Server Is Down` — неофициальный код состояния, специфичный для [Cloudflare](https://cloudflare.com).

Эта ошибка возникает, когда исходный веб-сервер отклоняет соединения от Cloudflare.
