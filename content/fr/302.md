## Description

Le code de statut HTTP `302 Found` indique que la ressource demandée a été temporairement déplacée vers l'URL indiquée par l'en-tête `Location`. Un navigateur redirige vers cette page, mais les moteurs de recherche ne mettent pas à jour leurs liens vers la ressource.

Il est recommandé de définir le code `302` uniquement comme réponse aux méthodes `GET` ou `HEAD` et d'utiliser `307 Temporary Redirect` à la place, car le changement de méthode est explicitement interdit dans ce cas.

Dans les cas où vous souhaitez que la méthode utilisée soit changée en `GET`, utilisez `303 See Other` à la place.
