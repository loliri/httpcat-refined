## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

Cloudflare 错误 `1008 Access Denied (ASN banned)` 表示与访问者 IP 地址关联的自治系统号（ASN）已被封禁。ASN 代表由单一组织（如 ISP 或托管服务商）管理的一组 IP 前缀。
