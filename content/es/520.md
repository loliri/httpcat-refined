## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

HTTP `520 Web Server Returned an Unknown Error` es un código de estado no oficial específico de [Cloudflare](https://cloudflare.com).

El error `520` ocurre cuando el servidor web de origen devuelve a Cloudflare una respuesta vacía, desconocida o inesperada. Suele ser un error genérico para situaciones en las que el servidor de origen devuelve algo que Cloudflare no puede interpretar.

Causas comunes:

- El servidor de origen se cuelga o se reinicia durante la solicitud.
- El servidor de origen devuelve una respuesta que viola las especificaciones HTTP.
- Un firewall o software de seguridad en el origen bloquea la solicitud de Cloudflare.
