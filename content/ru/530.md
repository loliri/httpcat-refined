## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

HTTP `530 Site Frozen` — неофициальный код состояния, специфичный для Cloudflare и Pantheon.

#### Cloudflare

Cloudflare использует код состояния `530`, когда между пользователем и сервером сайта возникает проблема. В сопровождении более детального HTTP-кода состояния `1XXX` ошибка зависит от конкретной ситуации.

#### Pantheon

Pantheon возвращает код состояния `530 Site Frozen`, когда неактивный сайт Sandbox отклоняет HTTP-запросы. Это можно исправить, повторно активировав сайт в панели управления пользователя Pantheon.
