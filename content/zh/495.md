## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP `495 SSL Certificate Error` 是 nginx 专用的非官方客户端错误状态码，由服务器返回，表示客户端证书验证过程中发生了错误。
