## Description

Le code de statut HTTP `423 Locked` indique que la ressource ciblée est verrouillée, ce qui signifie qu'elle ne peut pas être accédée. Son contenu doit contenir des informations au format XML de WebDAV.

<aside class="info"><strong>Note :</strong> La capacité à verrouiller une ressource est spécifique à certains serveurs WebDAV. Les navigateurs accédant à des pages web ne rencontreront jamais ce code de statut.</aside>
