## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

HTTP `527 Railgun Error` — неофициальный код состояния, специфичный для [Cloudflare](https://cloudflare.com).

Ошибка `527` указывает на прерванное соединение между Cloudflare и сервером Railgun исходного сервера. Railgun — технология оптимизации WAN, разработанная Cloudflare для сжатия и ускорения доставки динамического контента.

Cloudflare прекратил поддержку продукта Railgun в 2023 году, поэтому этот код ошибки носит преимущественно исторический характер.
