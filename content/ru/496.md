## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

HTTP `496 SSL Certificate Required` — неофициальная ошибка клиента, специфичная для nginx, возвращаемая сервером, когда клиент не предоставил требуемый сертификат.
