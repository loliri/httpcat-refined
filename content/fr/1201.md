## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

L'erreur Cloudflare `1201 Origin returned malformed HTTP` se produit lorsque le serveur web d'origine renvoie une réponse HTTP qui n'est pas conforme à la spécification HTTP. Cloudflare ne peut pas analyser ou relayer une réponse malformée au client.
