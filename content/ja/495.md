## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

HTTP `495 SSL Certificate Error` は nginx 固有の非公式クライアントエラーステータスコードで、クライアント証明書の検証中にエラーが発生したことを示します。
