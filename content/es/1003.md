## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1003 Direct IP access not allowed` ocurre cuando un visitante intenta acceder directamente a una dirección IP de Cloudflare en lugar de a través de un nombre de dominio. Cloudflare no permite el acceso directo por IP a su infraestructura.
