## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

HTTP `494 Request Header Too Large` は nginx 固有の非公式ステータスコードで、クライアントのリクエストヘッダーが nginx 設定の `large_client_header_buffers` ディレクティブで設定された制限を超えていることを示します。
