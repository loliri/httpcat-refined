## Description

<aside class="warning"><strong>Non officiel :</strong> Ce code de statut HTTP n'est spécifié par aucun RFC.</aside>

Le code de statut HTTP `450 Blocked by Windows Parental Controls` est un code non officiel spécifique à Microsoft, renvoyé par le serveur pour indiquer que le contrôle parental est activé et bloque l'accès à la ressource demandée.
