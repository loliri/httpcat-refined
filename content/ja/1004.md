## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

Cloudflare エラー `1004 Host not configured to serve web traffic` は、ドメインが Cloudflare に存在するものの、Web トラフィックを提供するためのホスティング設定がされていない場合に発生します。ドメインオーナーは Cloudflare の DNS 管理のみを利用しており、Web ホスティングが設定されていない可能性があります。適切なオリジンサーバーの設定が必要です。
