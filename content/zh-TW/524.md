## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

HTTP `524 A Timeout Occurred` 是 [Cloudflare](https://cloudflare.com) 專用的非官方狀態碼。

`524` 錯誤表示 Cloudflare 已成功與來源 Web 伺服器建立 TCP 連線，但來源伺服器在 100 秒的連線逾時內未返回 HTTP 回應。

這與 `522 Connection Timed Out` 不同，後者發生在 TCP 連線本身無法建立的情況下。
