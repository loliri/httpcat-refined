## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP `420 Method Failure` / `420 Enhance Your Calm` 是一个非官方状态码，由服务器返回以表示客户端错误。Spring Framework 用它表示方法执行失败，Twitter 则用它表示客户端因请求过于频繁而被限速。

#### Spring Framework

Spring Framework 发送 `420 Method Failure` 时，表示某个方法执行失败。此状态码已被 Spring 废弃，不再使用。

#### Twitter

在 Twitter 搜索与趋势 API 1.0 版本中，`420 Enhance Your Calm` 用于表示客户端因请求过于频繁而被限速，目的是通过限制客户端在特定时间内的请求次数来缓解流量压力。随着 Twitter API 标准版 v1.1 的发布，此状态码已被废弃，改用官方的 `429 Too Many Requests`。
