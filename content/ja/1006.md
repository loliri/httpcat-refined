## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

Cloudflare エラー `1006 Access Denied (IP banned)` は、Cloudflare 自身が訪問者の IP アドレスを悪意のある活動を理由に禁止している場合に発生します。この禁止措置は Cloudflare のセキュリティシステムによって自動的に適用されます。正当なユーザーがこのエラーを受け取った場合は、Cloudflare のサポートに問い合わせてください。
