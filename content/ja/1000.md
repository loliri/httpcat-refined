## 説明

<aside class="warning"><strong>非公式：</strong>この HTTP ステータスコードはいかなる RFC でも指定されていません。</aside>

Cloudflare エラー `1000 DNS points to prohibited IP` は、Cloudflare の DNS リゾルバーがドメインの DNS レコードを確認した際、そのレコードが Cloudflare 自身の IP アドレスを指していることを検出した場合に発生します。これにより、Cloudflare がトラフィックを自分自身に転送するループが生じてしまいます。この問題を解決するには、オリジンサーバーの正しい IP アドレスを DNS レコードに設定してください。
