## Descripción

<aside class="warning"><strong>No oficial:</strong> Este código de estado HTTP no está especificado por ningún RFC.</aside>

El error de Cloudflare `1001 DNS resolution error` ocurre cuando Cloudflare no puede resolver el nombre de host del servidor web de origen. Esto suele significar que el registro DNS del origen no existe o está mal configurado.
