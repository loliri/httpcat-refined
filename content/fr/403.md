## Description

Le code de statut HTTP `403 Forbidden` indique que le serveur comprend la requête mais refuse de l'autoriser.

Ce statut est similaire à `401`, mais pour le code de statut `403 Forbidden`, la ré-authentification ne fait aucune différence. L'accès est lié à la logique de l'application, comme des droits insuffisants sur une ressource.
