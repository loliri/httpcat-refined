## Description

Le code de statut HTTP `410 Gone` indique que l'accès à la ressource cible n'est plus disponible sur le serveur d'origine et que cette condition est susceptible d'être permanente.

Si vous ne savez pas si cette condition est temporaire ou permanente, un code de statut `404` doit être utilisé à la place.

<aside class="info"><strong>Note :</strong> Une réponse 410 est mise en cache par défaut.</aside>
