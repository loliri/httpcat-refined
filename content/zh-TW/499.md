## 描述

<aside class="warning"><strong>非官方：</strong>此 HTTP 狀態碼未由任何 RFC 指定。</aside>

HTTP `499` 是非官方狀態碼，ArcGIS 使用 `499 Token Required`，nginx 使用 `499 Client Closed Request`。

#### ArcGIS

收到 `499 Token Required` 狀態碼時，表示伺服器期望 HTTP 請求中包含令牌，但未提交。

#### nginx

收到 `499 Client Closed Request` 狀態碼時，表示客戶端在伺服器完成任務之前提前關閉了 HTTP 連線，因此無法傳送最終回應。此狀態碼僅出現在 nginx 日誌中。
