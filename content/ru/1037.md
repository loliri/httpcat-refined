## Описание

<aside class="warning"><strong>Неофициально:</strong> Этот код состояния HTTP не определён ни одним RFC.</aside>

Ошибка Cloudflare `1037 Invalid request rewrite (invalid header name)` возникает, когда правило преобразования Cloudflare пытается установить или изменить HTTP-заголовок с использованием недопустимого имени заголовка. Имена заголовков должны соответствовать спецификациям HTTP.
