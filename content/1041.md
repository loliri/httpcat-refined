## Description

<aside class="warning"><strong>Unofficial:</strong> This HTTP status code is not specified by any RFC.</aside>

Cloudflare error `1041 Invalid request rewrite (header modification rejected)` occurs when a Cloudflare Transform Rule attempts to modify a protected or read-only HTTP header that cannot be changed.
